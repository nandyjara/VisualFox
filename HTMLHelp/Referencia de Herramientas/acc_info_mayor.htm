Módulo: Contabilidad
Informe: Mayor de Cuenta

Definición

Este informe genera un reporte de los movimientos de una cuenta de detalle, este informe presenta el tipo de movimiento, el tipo de asiento del cual se generó el movimiento, aquí se presenta todos los detalles de los asientos en los que estuvo involucrada la cuenta.

Acceso

Para acceder a este informe:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el submenu Contabilidad y desde la sección Informes seleccione el comando Mayor de Cuentas.

Filtros y vínculos

Elemento

Descripción

Código

 Cuadro desplegable para seleccionar o localizar una cuenta. Puede escribir el código de cuenta, presionar las teclas ALT+FLECHA_ABAJO para desplegar una lista de cuentas simple con búsqueda incremental por código; o puede presionar CTRL+FLECHA_ABAJO para abrir un cuadro de búsqueda de cuentas por filtro.

Fechas

Seleccione el rango de fechas en que desea que se filtre la información

Asiento

Este botón le permite hacer un vinculo al Documento Asientos de Diario, para observar todo el detalle de un asiento específico debe seleccionar la fila donde se encuentra el mismo y dar un Click en el botón Asiento>>

Comentarios

El informe de Mayor de Cuentas le presenta una línea de encabezado con el Saldo Anterior que le presenta el saldo del movimiento anterior a la fecha inicial seleccionada por el usuario y al pie de todas las líneas de detalle el Saldo final que presenta el saldo a la fecha final seleccionada por el usuario.
