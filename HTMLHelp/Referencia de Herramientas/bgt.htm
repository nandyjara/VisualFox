Módulo: Presupuesto
Introducción

El Módulo de Presupuesto proporciona las herramientas para la elaboración de PLANES PRESUPUESTARIOS que facilitarán la generación de Informes de Presupuesto comparativos.  El Módulo de Presupuesto no genera transacciones contables de ningún tipo; este módulo utiliza la información generada para compararla contra las estimaciones mensuales definidas por el usuario. 

Para la elaboración de un Plan Presupuestario, DOBRA® proporciona un Fichero de Presupuestos que facilita la creación de varias estructuras jerárquicas denominadas PRESUPUESTOS formadas de elementos denominados CÓDIGOS PRESUPUESTARIOS.   Luego, esta estructura jerárquica se aplica en uno o varios tipos de Informes de Presupuesto en donde los datos se pivotan mensualmente para cada código de presupuesto; logrando además, mostrarse informes comparativos con el movimiento Real.  Mediante este mecanismoo, DOBRA®, permite crear varios tipos de Presupuestos: FLUJOS, BALANCES, GANANCIAS O PERDIDAS, etc.

Comparación

Para poder realizar comparaciones, a cada Código Presupuestario se le puede asignar una o mas cuentas o grupoas de cuentas desde el Plan de Cuentas de Contabilidad.  Al momento de elaborar informes de Presupuesto, DOBRA puede calcular los importes o saldos de cada cuenta y sumarlos al Código Presupuestario, de esta manera se puede comparar los valores reales contables contra los valores mensuales presupuestados.  En la siguiente figura se muestra un ejemplo de la asignación de cuentas.
