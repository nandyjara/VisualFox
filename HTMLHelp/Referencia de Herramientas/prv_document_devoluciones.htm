Módulo de Proveeduria
Documento: Devoluciones

Definición

La Devolución de Compra sirve para registrar los productos que se devuelven al Proveedor porque no cumplen con las especificaciones solicitadas o porque están averiados.

La Devolución de Compra representa un Débito en la cuenta de la empresa con el Proveedor, por esta razón, el usuario debe aplicar el monto de la devolución a una o varias de las deudas que la empresa tiene con el Proveedor.

El valor que el usuario puede escribir en la columna cantidad de la Lista de Productos, no puede ser mayor a la cantidad facturada de la Factura de Compra.  La cantidad facturada en la Factura de Compra se puede ver reducida con cada Devolución de Compra efectuada, de esta manera, la próxima vez que el usuario registre una Devolución de Compra la cantidad facturada será menor.

Acceso

Para acceder a esta opción, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el Submenú Proveduria y desde la sección Documento seleccione el comando Devoluciones.

Elementos

 

Elemento

Descripción
Buscar Sirve para buscar el documento por número.
Estado El cuadro estado es de "solo lectura" y muestra información acerca del estado actual del documento Ej: ANULADO, APROBADO, ETC.
ID El cuadro ID es de "solo lectura" y muestra el número de identificación relacional del documento que es generado por el sistema.
Número El cuadro Numero es de "solo lectura" y muestra número secuencial del documento que es generado por el sistema.
Asiento El cuadro Asiento es de "solo lectura" y muestra el número del asiento contable asociado al documento y que es generado por el sistema.
Tipo El cuadro Tipo es de "solo lectura" y muestra el Tipo de transacción.
Fecha El cuadro Fecha es "editable" y muestra fecha de la transacción.
No. de Factura El cuadro No. de factura sirve para ingresar el número de factura sobre la cual se hará la devolución.
División

El cuadro de División sirve para seleccionar la división o línea de negocio a la que pertenece el documento.
Proveedor El cuadro de Proveedor es de "solo lectura" y muestra el nombre del proveedor y que es generado por el sistema.
Divisas El cuadro divisas es de "Sólo lectura" y sirve para visualizar la divisa actual.
Cambio El cuadro Cambio es de "Sólo lectura" y sirve para visualizar el tipo de cambio.
Detalle de Productos

La sección de Detalle de Productos está compuesto por una cuadrícula de productos que tiene las columnas de código, nombre, bodega, cantidad, empaque, unidades, precio, desee%, desc, total.
La sección de detalle de Productos sirve para visualizar y registrar los  productos para la devolución además contiene el subtotal, descuento, impuestos, total.
Nota que se Imprime en la Devolución

El cuadro observaciones es "editable" y sirve para ingresar una descripción completa de la transacción.

Crear

  1. Seleccione el comando Nuevo desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.
  2. El cursor se ubica en el cuadro No. Factura.  Escriba el número de la factura y luego presione la tecla TAB para mover el cursor a la sección de Detalle de Productos.

  3. Para registrar la devolución de los productos, haga click en () y aparecerá una ventana de diálogo en la cual podrá escribir una explicación más detallada acerca del producto para la devolución y presione la tecla TAB para mover el cursor a la siguiente celda. 

     

  4. Seleccione la bodega de donde los productos han sido facturados.  Use la tecla ALT+FLECHA_ABAJO para seleccionar.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.  

  5. Ingrese la cantidad  de la devolución del producto .  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.

  6. Seleccione el empaque, éste viene dado internamente según el producto de venta.  Use la tecla ALT+FLECHA_ABAJO para seleccionar.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.  

  7. Luego puede escribir una glosa con el detalle de la devolución.  Cuando este listo debe asegurarse que todos los valores estén calculados correctamente.

  8. Seleccione el comando Guardar desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.

  9. A continuación el sistema le mostrará con el nuevo número de documento asignado.
