Módulo de Transporte
Informe:  Informe de Ordenes (Detallado)

Definición

El Informe de Ordenes Detallado presenta datos de las ordenes de trabajo de una manera detallada con cada uno de los items que forman parte de cada una de las ordenes de trabajo filtradas, este informa presenta la información tanto a nivel genera por todos los clientes, como ordenes de trabajo por cliente.

Integración

No se puede realizar ningún tipo de transacción.

Acceso

Para acceder a esta opción, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el Submenú Transporte y desde la sección Informe seleccione el comando Informe de Ordenes (Detallado).

Elementos

Elemento

Descripción
Sirve para abrir el arbol de Ordenes: Por confirmar precios, Por confirmar camiones, Por confirmar estibas, Por confirmar llegada, LLegada confirmada, Facturado, Procesados
Sirve para abrir la cuadrícula inferior de información: Observaciones, Historia.
Filtros En la parte superior del informe encontramos varios cuadros para filtrar la información como: Fechas, Cliente, Estado, Cuadro de ubicación por Orden
Acción Botón para realizar acciones sobre las ordenes detalladas
Cuadrícula de información El informe presenta datos de: F, E, C, LL (si se registro Factura-Estibas-Camiones-Llegadas), Número, Fecha, Orden, Contenedor, Cliente, Vehículo, Chofer, Ruta, Peso, Volumen, Módulo, Est, Tipo, Clase, Estibadores, Estado 

Crear

Para crear una nuevo Informe, haga lo siguiente:

  1. Seleccionar el rango de fechas o cualquiera de los otros criterios que desea consultar y presione la tecla F5 o el botón consultar.

  2. Acontinuación el sistema le mostrará los datos de acuerdo a los criterios de busqueda seleccionados por el usuario.

Acción

El botón Acción le proporciona al usuario las siguientes opciones:

    Finalizar etapa 

    Con esta acción el usuario finaliza la etapa de la orden de trabajo en que se encuentra actualmente a la etapa indicada por el usuario.

    

  Confirmar Datos de Camiones 

    Con esta acción el usuario puede confirmar y editar los datos de camiones.

Confirmar Datos de Estibas 

    Con esta acción el usuario puede confirmar y editar los datos de estibas como el tipo de estiba, el trabajo a realizar, el número de estibadores.

Confirmar Llegada 

    Con esta acción el usuario puede confirmar y editar los datos de llegada de los camiones al destino de ruta.

 

Confirmar Precios 

    Con esta acción el usuario puede confirmar y editar los datos de precios por peso y volúmen como se lo realiza en el proceso asignación de precios.
