Módulo: RMA
Documento:  Envío de RMA al Proveedor

Definición

Aquí se registra el nombre, serie y el tipo de problema, los mismos que son enviados al proveedor para que sean canjeados o reparados siempre y cuando tenga la garantía.

Acceso

Para acceder a esta opción, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el Submenú RMA y desde la sección Documento seleccione el comando Envío de RMA al Proveedor.

Elementos

Elemento

Descripción
Buscar Sirve para buscar el documento por número.
Estado El cuadro estado es de "solo lectura" y muestra información acerca del estado actual del documento Ej: ANULADO, APROBADO, ETC.
ID El cuadro ID es de "solo lectura" y muestra el número de identificación relacional del documento que es generado por el sistema.
Número El cuadro Numero es de "solo lectura" y muestra número secuencial del documento que es generado por el sistema.
Asiento El cuadro Asiento es de "solo lectura" y muestra el número del asiento contable asociado al documento y que es generado por el sistema.
Tipo El cuadro Tipo es de "solo lectura" y muestra el Tipo de transacción.
Fecha El cuadro Fecha es "editable" y muestra fecha de la transacción.
Divisa El cuadro de divisa sirve para seleccionar la divisa ó visualizar ésta divisa según el tipo de cambio.
Cambio El cuadro Cambio es de "Sólo lectura" y sirve para visualizar el tipo de cambio.
Vendedor El cuadro de Vendedor sirve para seleccionar el vendedor quién realiza la factura de empleados.
Cliente El cuadro de cliente sirve para seleccionar el cliente a quién se realiza la factura de empleados.
No.Factura El cuadro de No. factura sirve para ingresar el número de la factura de empleados.
Término El cuadro de Términos sirve para seleccionar el plazo 15, 20, 45 días en que se realiza la factura de venta.
Nombre El cuadro Nombre es de "solo lectura" y muestra el nombre que es generado por el sistema.
RUC El cuadro de RUC sirve para ingresar el número de cédula del empleado.
No. Guía El cuadro de No. Guía sirve para ingresar el No. guía remisión del transporte.
Transporte El cuadro de Transporte sirve para seleccionar medio de transporte.
División El cuadro de División sirve para seleccionar la división o línea de negocio a la que pertenece el documento.
O. Venta El cuadro de O. Venta  es de "solo lectura" y muestra la orden de venta que es generado por el sistema.
Detalle de Productos La sección de Detalle de Productos está compuesto por una cuadrícula productos que tiene las columnas de Código, nombre, botón de detalle de productos, bodega, empaque, cantidad, valor, desc%, desc, total, extendido, imp  y además de los botones de comandos Nuevo y Quitar que sirve para agregar o quitar filas en la cuadrícula. La sección de detalle de Productos sirve para registrar los productos que van a ser facturados.
Pestaña Nota La pestaña Nota está compuesta por dos glosas nota que se imprime en la factura y comentario para uso exclusivo de la empresa . 
Pestaña Forma de Pagos La pestaña Forma de Pagos está compuesta por tipo, fecha, vencimiento, detalle, cuota y sirve para registrar las deudas por pagar de los clientes.

Crear

Para crear un nuevo Documento, haga lo siguiente:

  1. Seleccione el comando Nuevo desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.

  2. El cursor se ubica en el cuadro cliente.  Escriba el nombre del cliente y luego presione la tecla TAB para mover el el cursor al cuadro vendedor.

  3. El cursor se ubica en el cuadro vendedor.  Escriba el nombre del vendedor y luego presione la tecla TAB para mover el el cursor al cuadro No. factura.

  4. Ingrese el número de factura.  A continuación presione la tecla TAB para mover el cursor al cuadro término.

  5. Seleccione el término de pago al que corresponde la factura.  Puede usar las teclas ALT+FLECHA_ABAJO o CTRL+FLECHA_ABAJO.  A continuación presione la tecla TAB para mover el cursor al cuadro No. de guía.

  6. Ingrese el número de guía de remisión.  A continuación presione la tecla TAB para mover el cursor al cuadro transporte.

  7. Seleccione el transporte.  Puede usar las teclas ALT+FLECHA_ABAJO o CTRL+FLECHA_ABAJO.  A continuación presione la tecla TAB para mover el cursor al cuadro división.

  8. Seleccione la división a la que corresponde la factura.  Puede usar las teclas ALT+FLECHA_ABAJO o CTRL+FLECHA_ABAJO.  A continuación presione la tecla TAB para mover el cursor a la sección de detalle de productos .

  9. Para registrar los productos que serán descritos en la factura, se debe agregar nuevas filas en la cuadrícula de Detalle de Productos.  Para agregar una nueva fila haga un click en el botón () de la sección Detalle de Productos o presione la tecla ALT+N.  Aparece una nueva fila en la cuadrícula y el cursor se ubica en la primera celda de la fila.

  10. Seleccione el código del producto.  Use la tecla ALT+FLECHA_ABAJO y seleccione el producto.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.  

  11. Haga click en () y aparecerá una ventana de diálogo en la cual podrá escribir una explicación más detallada acerca del producto y presione la tecla TAB para mover el cursor a la siguiente celda. 

     

  12. Seleccione la bodega de donde los productos son facturados.  Use la tecla ALT+FLECHA_ABAJO para seleccionar.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.  

  13. Ingrese la cantidad del producto.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.

  14. Seleccione el empaque, éste viene dado internamente según el producto de venta.  Use la tecla ALT+FLECHA_ABAJO para seleccionar.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.  

  15. Ingrese el valor del descuento del producto.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.

  16. Luego de haber seleccionado todos los productos para la factura, ud.  Puede asignarle el valor del impuesto(IVA) a cada producto.  Haga click en () y el sistema se encargará de asignarle internamente el valor correspondiente del IVA a cada producto, para su calculo posterior. 

  17. Si desea ingresar una nuevo Detalle de Productos regrese al paso 9.  Si desea quitar una fila de la cuadrícula de productos, haga un click en la fila correspondiente y luego haga un click el botón () de la sección Detalle de Productos ó presione la tecla ALT+Q.  

  18. Seleccione la Pestaña Nota, aquí puede escribir una nota que se imprime en la factura y además de un comentario para uso exclusivo de la empresa. 
    Seleccione la Pestaña Forma de Pagos. Y luego haga click en el spiner (incrementador) ( ) donde se elige cuantas en cuotas desea repartir el pago.

  19. Haga click en el botón de comandos ( ).  A continuación Ud. Puede visualizar los pagos generados repartidos en cuotas iguales por el sistema, también podrá repartir valores manualmente si se desea.

  20. Seleccione el comando Guardar desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.

  21. A continuación el sistema le mostrará con el nuevo número de documento asignado.
