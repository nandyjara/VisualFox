Módulo de Trámites
Mantenimiento: Opciones

Definición

El Fichero de Opciones sirve para establecer parámetros iniciales para trabajar con el Módulo de Trámites. 

Acceso

Para acceder a esta opción, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el Submenú Trámites y desde la sección Mantenimiento seleccione el comando Opciones.

Elementos

Elemento

Descripción
Valor Max. para Pago en Efectivo

Se parametriza el valor máximo a realizar pagos en efectivo
Actualizar Bandeja de Trámites cada:

Campo para indicar cada que tiempo se actualiza automáticamente la Bandeja de Trámites, si no se especifica ningún valor el usuario debe presionar manualmente el botón actualizar de la Bandeja de Trámites.
Buzón PreDefinido para Bandeja de Comunicaciones

Campo para establecer el buzón que tendrá acceso a la bandeja de Comunicaciones
Etapa para validar doc. Completa Se selecciona la etapa donde se valida si la documentación está completa
Etapa para validar Partidas/Items Se selecciona etapa donde se validan las Partidas e Items
Honorario para garantías [PP]

Se especifica el honorario predeterminado para papeles previos.
Formato de Aplicación/Seguro

En este campo se selecciona el formato de impresión de la Aplicación/Seguro
Tiempo de Autorizaciones Salud/MAC Tiempo que demora en autorizar las dependencias estatales: Ministerio de Salud
Impuestos de Papeles Previos Porcentaje y valor a pagar por el trámite de papeles previos
Valor de Prima Mínimo Valor mínimo a pagar por la prima
Jornada laboral (Entrada/Salida) Jornada laboral de empledos
ID Etapa de Ciateite ID en el cual inicia la etapa CIATEITE
Etapa para grabar el ID Ciateite Etapa de T&T en la cual inicia la primera etapa CIATEITE
Indice de Oficina Etapa de inicio y etapa de fin para medir índices de Oficina
Indice de Tiempo para el Cliente Etapa de fin para medir índices de Oficina
Indice de Puerto Etapa de inicio y etapa de fin para medir índices de Puerto

Crear

Para crear una nueva Opción, haga lo siguiente:

  1. Debe ingresar los datos de los campos del fichero.

  2. Seleccione el comando Guardar desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.

Modificar

Para modificar una Opción, haga lo siguiente:

  1. Directamente ubicarse en el campo que desea modificar, realizar la edición y guardar el cambio.

Eliminar

No existe método de eliminación en este fichero.
