Módulo de Transporte
Informe:  Orden de Trabajo

Definición

El Informe de Orden de Trabajo presenta todas las ordenes de trabajos con sus respectivos estados (procesado, facturado, pendiente, anulado), este informa presenta la información tanto a nivel general por todos los clientes, como ordenes de trabajo por cliente.

Integración

No se puede realizar ningún tipo de transacción.

Acceso

Para acceder a esta opción, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el Submenú Transporte y desde la sección Informe seleccione el comando Orden de Transporte.

Elementos

Elemento

Descripción
Sirve para abrir la cuadrícula inferior de información: Detalles de viajes y Siniestros
Filtros En la parte superior del informe encontramos un filtro de cliente, para seleccionar el informe solo por cliente, si el usuario no selecciona un cliente el informe filtrará la información de todos los clientes
Acción Botón para realizar acciones sobre las ordenes de transporte
Documento Vinculo del informe al documento de orden de transporte
Cuadrícula de información El informe presenta datos de: Número, fecha, trámite, Cliente, Bultos, módulo, peso, volumen, transporte, estibada, total, estado.

Crear

Para consultar un nuevo informe:

  1. Seleccionar el cliente si así lo desea y presione la tecla F5 o el botón consultar.

  2. Acontinuación el sistema le mostrará los datos de acuerdo a los criterios de busqueda seleccionados por el usuario.

  3. Cada línea se presenta acompañado de botones que identifican los estados pendiente () y facturado ( ).

Acción

El botón Acción le proporciona al usuario las siguientes opciones:

    Confirmar datos del Vehículo

    Con esta acción el usuario puede confirmar y editar los datos del camión.

    Confirmar datos de Estibadores

    Con esta acción el usuario puede confirmar y editar los datos de estiba.

    Detalle de Viajes

    Con esta acción el usuario puede agregar observaciones a la cuadrícula de la parte inferior del informe llamada Detalle de viajes.

    Listo para facturar

    Esta acción no tiene ninguna pantalla, y lo que realizar es el cambio de pendiente a listo de facturar ().

    

    Siniestros

    Con esta acción el usuario puede agregar observaciones a la cuadrícula de la parte inferior del informe llamada Siniestros.
