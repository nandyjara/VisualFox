Módulo: Empleados
Informe: Saldos

Definición

El Informe de Saldos presenta un reporte de saldos basados en deudas pendientes. Además calcula un Total de Deudas.

Acceso

Para acceder a esta opción, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el Submenú Empleados y desde la sección Informe seleccione el comando Saldos.

Elementos


Elemento

Descripción
Divisa Cuadro desplegable de divisas.
Fecha Fecha que se va a filtrar la información para imprimir el reporte
Detalle Seleccionando un empleado y presionando este botón se creará un vínculo al documento de Deudas del Empleado.
Arbol Presionando este botón el usuario podrá filtrar empleados por Grupos y Zonas
Plantilla Se presenta Datos como: Código, Nombre, Deudas, Clase, ID
