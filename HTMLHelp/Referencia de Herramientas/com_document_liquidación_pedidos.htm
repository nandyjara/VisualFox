Módulo: Compras
Documento:  Liquidación de Importación

Definición

El Documento Liquidación de Importación permite realizar un informe para liquidar las notas de Importación. La liquidación ingresa la mercadería al Inventario.

Integración

Acceso

Para acceder a esta opción, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el Submenú compras y desde la sección Documento seleccione el comando Liquidación de Importación.

Elementos

Elemento

Descripción
Buscar Sirve para buscar el documento por número.
Estado El cuadro estado es de "solo lectura" y muestra información acerca del estado actual del documento Ej: ANULADO, APROBADO, ETC.
ID El cuadro ID es de "solo lectura" y muestra el número de identificación relacional del documento que es generado por el sistema.
Número El cuadro Numero es de "solo lectura" y muestra número secuencial del documento que es generado por el sistema.
No. Pedido El cuadro Asiento es de "solo lectura" y muestra el número del asiento contable asociado al documento y que es generado por el sistema.
Tipo El cuadro Tipo es de "solo lectura" y muestra el Tipo de transacción.
Fecha El cuadro Fecha es "editable" y muestra fecha de la transacción.
Divisa El cuadro de divisa sirve para seleccionar la divisa ó visualizar ésta divisa según el tipo de cambio.
Cambio El cuadro Cambio es de "Sólo lectura" y sirve para visualizar el tipo de cambio.
División

El cuadro de División sirve para seleccionar la división o línea de negocio a la que pertenece el documento.

Detalle de Gastos de Importación

La sección de Detalle de Gastos de Importación está compuesto por una cuadrícula de productos que tiene las columnas de fecha, número, tipo, detalle, divisa, valor, valor base.  La sección de detalle de Productos sirve para visualizar los gastos de importación de la empresa.
Detalle de los Productos Pedidos

La sección de Detalle de Productos Pedidos está compuesto por una cuadrícula de productos que tiene las columnas de código, nombre, cantidad, empaque, total, bodega, precio, total precio, costo total.  La sección de detalle de Productos sirve para visualizar los productos pedidos por la empresa.
Nota

El cuadro observaciones es "editable" y sirve para ingresar una descripción completa de la transacción.

Crear

Para crear un nuevo Documento, haga lo siguiente:

  1. Seleccione el comando Nuevo desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.

  2. El cursor se ubica en el cuadro No. Pedido. Escriba el escriba el número de pedido de la nota de pedido de importación y luego presione la tecla TAB para mover el el cursor al cuadro .

  3. Ingrese la fecha de la liquidación (dd/mm/aa) y presione la tecla TAB para mover el cursor al siguiente cuadro divisa.

  4. Seleccione la divisa de la liquidación.  Generalmente en la celda de divisa aparece seleccionada la divisa base del sistema, si no necesita cambiar la divisa entonces solo presione la tecla TAB para mover el cursor a la siguiente celda; en caso contrario, use la tecla ALT+FLECHA_ABAJO y seleccione la divisa.  A continuación presione la tecla TAB para mover el cursor a la siguiente cuadro.  

  5. Ingrese la división y presione la tecla TAB para mover el cursor a la sección detalle de productos pedidos.

  6. Seleccione el código del producto para la liquidación y presione la tecla TAB para mover el cursor a la siguiente celda.

  7. Seleccione el empaque del producto y presione la tecla TAB para mover el cursor a la siguiente celda.

  8. Seleccione la bodega donde corresponde el incremento de inventario, debe selecionar una bodegaa de la misma sucursal.  y presione la tecla TAB para mover el cursor a la siguiente celda.

  9. Seleccione el costo de la liquidación y presione la tecla TAB para mover el cursor al siguiente cuadro.

  10. Luego puede escribir una glosa con el detalle del comprobante de egreso de importación.  Luego de haber finalizado debe asegurarse que todos los valores estén correctamente calculados.

  11. Seleccione el comando Guardar desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.

  12. Acontinuación el sistema le mostrará con el nuevo número de documento asignado.
