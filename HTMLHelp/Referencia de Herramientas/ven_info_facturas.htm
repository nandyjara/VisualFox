Módulo de Ventas
Informe: Facturas de Venta

Definición

El Informe de Facturas le permite al usuario extraer un reporte de todas las Facturas hechas a Clientes por un periodo de tiempo.

Acceso

Para acceder a esta opción, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el Submenú Ventas y desde la sección Informe seleccione el comando Facturas de Venta.

Elementos

Elemento

Descripción
Buscar Sirve para buscar el documento por número.
Estado El cuadro estado es de "solo lectura" y muestra información acerca del estado actual del documento Ej: ANULADO, APROBADO, ETC.
ID El cuadro ID es de "solo lectura" y muestra el número de identificación relacional del documento que es generado por el sistema.
Número El cuadro Numero es de "solo lectura" y muestra número secuencial del documento que es generado por el sistema.
Asiento El cuadro Asiento es de "solo lectura" y muestra el número del asiento contable asociado al documento y que es generado por el sistema.
Tipo El cuadro Tipo es de "solo lectura" y muestra el Tipo de transacción.
Fecha El cuadro Fecha es de "editable" y muestra fecha de la transacción.
Divisa El cuadro de divisa sirve para seleccionar la divisa ó se visualiza la divisa según el tipo de cambio.
Cambio

El cuadro Cambio es de "Sólo lectura" y sirve para visualizar el tipo de cambio.
División

El cuadro de División sirve para seleccionar la división o línea de negocio a la que pertenece el documento
Cliente

El cuadro de Cliente sirve para seleccionar el cliente a quién se le hace la orden de venta.
Vendedor

El cuadro de Vendedor sirve para seleccionar el vendedor quién efectúa la orden de venta.
Términos

El cuadro de Términos sirve para seleccionar el plazo 15, 20, 45 días en que se realiza la orden de venta.
Detalle de Productos y Servicios

La sección de detalle de Productos y Servicios está compuesta por los cuadros código, nombre, cantidad, empaque, total, lista, precio, desc%, desc, extendido, imp.; Estos cuadros son editables y sirven para ingresar los datos de los productos. Además cuenta con el botón de comando nuevo y quitar el cual sirve para quitar o agragar una nueva fila en la cuadrícula, además del cuadro Pares.
Nota que se Imprime en la Orden

Este cuadro es "editable" y sirve para ingresar una descripción impresa de la transacción.
Comentario Interno

Este cuadro es "editable" y sirve para ingresar una comentario  de la transacción para uso interno.
