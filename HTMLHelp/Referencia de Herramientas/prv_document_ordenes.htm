Módulo de Proveeduria
Documento: Ordenes de compra

Definición

La Orden de Compra sirve para registrar los productos para uso interno de la Empresa solicitados al Proveedor.

Acceso

Para acceder a esta opción, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el Submenú Proveduria y desde la sección Documento seleccione el comando Ordenes de Compra.

Elementos

Elemento

Descripción
Buscar Sirve para buscar el documento por número.
Estado El cuadro estado es de "solo lectura" y muestra información acerca del estado actual del documento Ej: ANULADO, APROBADO, ETC.
ID El cuadro ID es de "solo lectura" y muestra el número de identificación relacional del documento que es generado por el sistema.
Número El cuadro Numero es de "solo lectura" y muestra número secuencial del documento que es generado por el sistema.
Asiento El cuadro Asiento es de "solo lectura" y muestra el número del asiento contable asociado al documento y que es generado por el sistema.
Tipo El cuadro Tipo es de "solo lectura" y muestra el Tipo de transacción.
Fecha El cuadro Fecha es de "editable" y muestra fecha de la transacción.
Divisas El cuadro de divisa sirve para seleccionar la divisa ó se visualiza la divisa según el tipo de cambio.
Cambio El cuadro Cambio es de "Sólo lectura" y sirve para visualizar el tipo de cambio.
División El cuadro de División sirve para seleccionar la división o línea de negocio a la que pertenece el documento
Proveedor El cuadro Proveedor sirve para seleccionar el proveedor a quién se le hace el pedido.
Solicitud Este cuadro sirve para ingresar el número de solicitud de la orden.
Término El cuadro Término es "editable" y sirve para seleccionar la forma de pago pueden ser  a 5, 15, 20 días.
Detalle de orden de compra La sección de Detalle de orden de compra está compuesta por los cuadros código, nombre, cantidad, empaque, total, precio, desc%, desc, extendido; Estos cuadros son editables y sirven para ingresar los productos que se hacen en el pedido. Además cuenta con el botón de comando nuevo y quitar el cual sirve para quitar o agregar una nueva fila en la cuadrícula.
Nota que se Imprime en la Orden Este cuadro es "editable" y sirve para ingresar una descripción impresa de la transacción.

Crear

Para crear una nuevo Documento, haga lo siguiente:

  1. Seleccione el comando Nuevo desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.
  2. El cursor se ubica en el cuadro Proveedor.  Escriba el nombre del proveedor y luego presione la tecla TAB para mover el el cursor al cuadro solicitud.
  3. Ingrese el número de solicitud.  Acontinuación presione la tecla TAB para mover el cursor al cuadro término.
  4. Seleccione el término de pago al que corresponde la factura.  Puede usar las teclas ALT+FLECHA_ABAJO o CTRL+FLECHA_ABAJO.  Acontinuación presione la tecla TAB para mover el cursor al cuadro seguro.
  5. seleccione la división o línea del negocio.

  6. Para registrar los productos que van en la orden de compra, se debe agregar nuevas filas en la cuadrícula de Detalle de la orden.  Para agregar una nueva fila haga un click en el botón () de la sección Detalle de la orden o presione la tecla ALT+N.  Aparece una nueva fila en la cuadrícula y el cursor se ubica en la primera celda de la fila.

  7. Seleccione el código del producto.  Use la tecla ALT+FLECHA_ABAJO y seleccione el producto.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.  

  8. Ingrese la cantidad del producto.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.

  9. Seleccione el empaque, éste viene dado internamente según el producto de venta.  Use la tecla ALT+FLECHA_ABAJO para seleccionar.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.  

  10. Ingrese el precio del producto.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.

  11. Si desea ingresar una nuevo Detalle de orden de compra regrese al paso 12.  Si desea quitar una fila de la cuadrícula de orden de compra, haga un click en la fila correspondiente y luego haga un click el botón () de la sección Detalle de orden de compra ó presione la tecla ALT+Q.  

  12. Luego puede escribir una glosa con el detalle de la orden de compra.  Cuando este listo debe asegurarse que todos los valores estén calculados correctamente.

  13. Seleccione el comando Guardar desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.

  14. Acontinuación el sistema le mostrará con el nuevo número de documento asignado.
