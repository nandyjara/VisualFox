Módulo: Clientes
Documento: Cotizaciones

Definición

El Documento de Cotizaciones sirve para crear, modificar o eliminar cotizaciones solicitadas por los Clientes.

Acceso

Para acceder a esta opción, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el Submenú Clientes y desde la sección Documento seleccione el comando Cotizaciones.

Elementos

Elemento

Descripción
Buscar Sirve para buscar el documento por número.
Estado El cuadro estado es de "solo lectura" y muestra información acerca del estado actual del documento Ej: ANULADO, APROBADO, ETC.
ID El cuadro ID es de "solo lectura" y muestra el número de identificación relacional del documento que es generado por el sistema.
Número El cuadro Numero es de "solo lectura" y muestra número secuencial del documento que es generado por el sistema.
Tipo El cuadro Tipo es de "solo lectura" y muestra el Tipo de transacción.
Fecha El cuadro Fecha es "editable" y muestra fecha de la transacción.
Divisas El cuadro de lista de división sirve para seleccionar la división o línea de negocio a la que pertenece el documento
Cambio El cuadro Cambio es de "solo lectura" y muestra el dolar, osea la moneda o tipo de cambio actual.
Cliente

El cuadro de Cliente sirve para seleccionar el nombre del cliente a quien se le hace la cotización. 
Vendedor

El cuadro Vendedor sirve para seleccionar el nombre del vendedor.
Atención

El cuadro Atención sirve para escribir el nombre de la persona que atendió al cliente a quien se le hace la cotización.
Validez

El cuadro Validez sirve para escribir el nombre de la persona que atendió al cliente a quien se le hace la cotización.
División El cuadro División sirve para escribir la división a la que la persona petenece.
Término El cuadro Término sirve para seleccionar la forma de pago de contado, a 15 días, a 12 meses, etc.
Detalle de Productos

La sección de Detalle de Productos está compuesta por una cuadrícula de productos que tiene las columnas de código, nombre, cantidad, empaque, total, precio, desc%, desc, extendido y además de los botones de comandos Nuevo y Quitar que sirve para agregar o quitar filas en la cuadrícula. La sección de Detalle de Producto sirve para registrar las características del producto.
La sección de Detalle de Rubros de Crédito cuenta además con el suctotal, descuentos, impuestos, total.
Glosa

El cuadro observaciones es "editable" y sirve para ingresar una descripción completa de la transacción.

Crear

Para crear una nueva Cuenta, haga lo siguiente:

  1. Seleccione el comando Nuevo desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.
  2. El cursor se ubica en el cuadro divisa.  Seleccione la divisa del pago que se está realizando.  Generalmente en el cuadro de divisa aparece seleccionada la divisa base del sistema, use la tecla ALT+FLECHA_ABAJO y seleccione la divisa.  A continuación presione la tecla TAB para mover el cursor al cuadro cliente.  
  3. Seleccione el cliente.  Puede usar las teclas ALT+FLECHA_ABAJO o CTRL+FLECHA_ABAJO.  A continuación presione la tecla TAB para mover el cursor al cuadro vendedor.
  4. Seleccione el vendedor, Puede usar las teclas ALT+FLECHA_ABAJO o CTRL+FLECHA_ABAJO.  A continuación presione la tecla TAB para mover el cursor al cuadro de atención.

  5. Ingrese el nombre de la persona quien atendió al cliente y presione la tecla TAB para mover el cursor al siguiente cuadro de validez.

  6. Seleccione la validez de la cotización,  Puede usar las teclas ALT+FLECHA_ABAJO o CTRL+FLECHA_ABAJO.  A continuación presione la tecla TAB para mover el cursor al cuadro de término.

  7. Seleccione el término ó la forma de pago que puede ser de contado, a 12 meses, 15, 20 dias, etc;  Puede usar las teclas ALT+FLECHA_ABAJO o CTRL+FLECHA_ABAJO.  A continuación presione la tecla TAB para mover el cursor a la sección de Detalle de Productos.

  8. Para registrar las caractrísticas de los productos, se debe agregar nuevas filas en la cuadrícula de detalle de productos.  Para agregar una nueva fila haga un click en el botón () de la sección detalle de productos o presione la tecla ALT+N.  Aparece una nueva fila en la cuadrícula y el cursor se ubica en la primera celda de la fila.

  9. Seleccione el código del producto y presione la tecla TAB para mover el cursor a la siguiente celda.

  10. Haga click en () y aparecerá una ventana de diálogo en la cual podrá escribir una explicación más detallada acerca del producto que irá adjunta a la cotización y presione la tecla TAB para mover el cursor a la siguiente celda. 

     

  11. El stock disponible se visualiza automáticamente; esta celda es "editable".

  12. Ingrese la cantidad y presione la tecla TAB para mover el cursor a la siguiente celda.

  13. Seleccione el empaque que viene dado internamente según el producto en venta y presione la tecla TAB para mover el cursor a la siguiente celda.

  14. Seleccione la lista, forma o tipo de pago en que se ha adquirido el producto.  Cada producto puede ser adquirido de contado, a crédito, precio de distriduidor, etc;  luego presione la tecla TAB para mover el cursor a la siguiente celda.

  15. Ingrese el descuento y presione la tecla TAB para mover el cursor a la siguiente celda.

  16. Luego de haber seleccionado todos los productos para la cotización, ud.  Puede asignarle el valor del IVA a cada producto.  Haga click en () y el sistema se encargará de asignarle internamente el valor correspondiente del IVA a cada producto, para su calculo posterior. 

  17. Si desea ingresar un nuevo producto regrese al paso 7.  Si desea quitar una fila de la cuadrícula, haga un click en la fila correspondiente y luego haga un click el botón () de la sección Detalle de Productos ó presione la tecla ALT+Q.  

  18. Luego puede escribir una glosa con el detalle de la cotización.  Cuando este listo debe asegurarse que no exista ningún error en los totales calculado por el sistema.

  19. Seleccione el comando Guardar desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.

  20. A continuación el sistema le mostrará con el nuevo número de documento asignado.
