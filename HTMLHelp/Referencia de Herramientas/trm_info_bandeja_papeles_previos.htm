Módulo de Trámites
Informe:  Bandeja Papeles Previos

Definición

El Informe de bandeja de papeles previos presenta opciones de visualización y acciones que pueden ser aplicadas a los trámites de PP.

Integración

No se puede realizar ningún tipo de transacción.

Acceso

Para acceder a esta opción, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el Submenú Trámites y desde la sección Informe seleccione el comando Bandeja de papeles previos.

Elementos

Elemento

Descripción
Sirve para abrir el arbol de estados de Ordenes: Pendientes, Facturar, Urgente, Puerto
Sirve para abrir la cuadrícula inferior de información: Observaciones, Historia, Reembolso .
Filtros En la parte superior del informe encontramos varios cuadros para filtrar la información de ordenes como: Fechas, Cliente, Oficial de cuenta, Tipo de Trámite, Etapa, Número de Orden y Pedido de cliente
Acción Botón para realizar acciones sobre las ordenes de trámite
Documento Vinculo del informe al documento de trámites

Crear

Para consultar un nuevo informe:

  1. Seleccionar el rango de fechas o cualquiera de los otros criterios que desea consultar y presione la tecla F5 o el botón consultar.

  2. Acontinuación el sistema le mostrará los datos de acuerdo a los criterios de busqueda seleccionados por el usuario.

  3. Los datos se presentan en diferentes colores de acuerdo a las etapas, esto lo configura el usuario en el fichero de etapas del menú mantenimiento del módulo de trámites.

Acción

El botón Acción le proporciona al usuario las siguientes opciones:

    Regresar

    Regresa el trámite de la etapa actual a la anterior.

   

   Finalizar etapa

    Finaliza la etapa actual para continuar con la siguiente, el usuario debe notificar a un buzón del evento.


   Ordenes de Pago

Con esta acción el usuario puede realizar una cancelación de un  producto (reembolso) que será facturado al cliente, tiene la opción de enviar a un buzón.

 Ordenes de Pago Masivo

Con esta acción el usuario puede realizar una cancelación masiva de productos (reembolso) que será facturado a los clientes, tiene la opción de enviar a un buzón.

  Observaciones

Esta acción se utiliza para registrar alguna observación a la orden de trámites.

 Listo para Facturar

Con esta acción el usuario cambia la etapa actual de la orden a Listo para Facturar, no se presenta ninguna pantalla adicional.

Urgente

Con esta acción el usuario cambia la etapa actual de la orden a Urgente, no se presenta ninguna pantalla adicional.

 Delegar a Importación

Con esta acción el usuario delega el trámite al departamento de Importación.

Documentación Completa

Con esta acción el usuario establece que se tiene documentación completa de la orden de trámites.
