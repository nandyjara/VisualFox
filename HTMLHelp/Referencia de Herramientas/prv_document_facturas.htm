Módulo de Proveeduria
Documento: Facturas de Compra

Definición

La Factura de Orden de Compra sirve para registrar la Factura de Compra emitida por el Proveedor por los productos que ya fueron recibidos en la empresa.

Acceso

Para acceder a esta opción, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el Submenú Proveduria y desde la sección Documento seleccione el comando Facturas de Compra.

Elementos

Elemento

Descripción
Buscar Sirve para buscar el documento por número.
Estado El cuadro estado es de "solo lectura" y muestra información acerca del estado actual del documento Ej: ANULADO, APROBADO, ETC.
ID El cuadro ID es de "solo lectura" y muestra el número de identificación relacional del documento que es generado por el sistema.
Número El cuadro Numero es de "solo lectura" y muestra número secuencial del documento que es generado por el sistema.
Asiento El cuadro Asiento es de "solo lectura" y muestra el número del asiento contable asociado al documento y que es generado por el sistema.
Tipo El cuadro Tipo es de "solo lectura" y muestra el Tipo de transacción.
Fecha El cuadro Fecha es de "editable" y muestra fecha de la transacción.
Divisas El cuadro de divisa sirve para seleccionar la divisa ó se visualiza la divisa según el tipo de cambio.
Cambio El cuadro Cambio es de "Sólo lectura" y sirve para visualizar el tipo de cambio.
Proveedor El cuadro Proveedor sirve para seleccionar el proveedor a quién se le hace el pedido.
Fact. Prov El cuadro factura de provedor sirve para ingresar el número de factura del proveedor.
Término El cuadro Término es "editable" y sirve para seleccionar la forma de pago pueden ser  a 5, 15, 20 días.
O. Compra El cuadro de O. Compra  es de "solo lectura" y muestra la orden de compra que es generado por el sistema.
Detalle de Productos La sección de Detalle de Prosuctos está compuesta por los cuadros código, nombre, bodega, cantidad, empaque, total, precio, desc%, desc, extendido; Estos cuadros son editables y sirven para ingresar los productos que se hacen en el pedido. Además cuenta con el botón de comando nuevo y quitar el cual sirve para quitar o agregar una nueva fila en la cuadrícula.
Nota que se Imprime en la Orden Este cuadro es "editable" y sirve para ingresar una descripción impresa de la transacción.

Crear

  1. Seleccione el comando Nuevo desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.
  2. El cursor se ubica en el cuadro Proveedor.  Escriba el nombre del proveedor y luego presione la tecla TAB para mover el el cursor al cuadro Fact.Prov.

  3. Ingrese la factura del proveedor.  Acontinuación presione la tecla TAB para mover el cursor al cuadro término.

  4. Seleccione el témino de pago al que corresponde la factura.  Puede usar las teclas ALT+FLECHA_ABAJO o CTRL+FLECHA_ABAJO.  Acontinuación presione la tecla TAB para mover el cursor al cuadro división.

  5. Seleccione la división a la que corresponde la factura.  Puede usar las teclas ALT+FLECHA_ABAJO o CTRL+FLECHA_ABAJO.  Acontinuación presione la tecla TAB para mover a la sección de Detalle de Productos.

  6. Para registrar los productos que serán descritos en la factura, se debe agregar nuevas filas en la cuadrícula de Detalle de Productos.  Para agregar una nueva fila haga un click en el botón () de la sección Detalle de Productos o presione la tecla ALT+N.  Aparece una nueva fila en la cuadrícula y el cursor se ubica en la primera celda de la fila.

  7. Seleccione el código del producto.  Use la tecla ALT+FLECHA_ABAJO y seleccione el producto.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.  

  8. Seleccione la bodega de donde aumentan los productos.  Use la tecla ALT+FLECHA_ABAJO para seleccionar.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.  

  9. Ingrese la cantidad del producto.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.

  10. Seleccione el empaque, éste viene dado internamente según el producto de venta.  Use la tecla ALT+FLECHA_ABAJO para seleccionar.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.  

  11. Ingrese el valor del descuento del producto.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.

  12. Si desea ingresar una nuevo Detalle de Productos regrese al paso 6.  Si desea quitar una fila de la cuadrícula de productos, haga un click en la fila correspondiente y luego haga un click el botón () de la sección Detalle de Productos ó presione la tecla ALT+Q.  

  13. Luego puede escribir una glosa con el detalle de la factura.  Cuando este listo debe asegurarse que todos los valores estén calculados correctamente.

  14. Seleccione el comando Guardar desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.

  15. Acontinuación el sistema le mostrará con el nuevo número de documento asignado.
