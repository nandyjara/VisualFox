Módulo Académico
Procesos - Matrícula

Explicación

El Proceso de Matriculación sirve para registrar un estudiante en el nuevo Año y Periodo Lectivo, de tal manera que pueda ser tomado en cuenta para los diferentes procesos Financieros y Académicos en el Sistema.

Para matricular un estudiante, se requiere la siguiente información:

1 Año Lectivo El Año actual, generalmente este dato es proporcionado por el sistema basándose en la Fecha actual del Servidor.
2 Periodo Es un número que indica el periodo para el cual se matricula un estudiante.   generalmente para el caso de escuelas y colegios este valor siempre será 1 pero para el caso de Institutos y Universidades, este valor puede variar dependiendo si el año lectivo está compuesto de semestres o trimestres.  Lo más común es trabajar con 1 o 2 para Institutos y Universidades.
3 Nivel Orgánico Representa el Nivel en el que el estudiante estará ubicado para el año lectivo actual; es decir, dependiendo de la complejidad del Esquema Orgánico de la Institución, el estudiante debe ser asignado a un Paralelo.
4 Tipo de Beca Identifica el Tipo de Beca: NORMAL, DIFERENCIADA, etc. que el estudiante ha solicitado y que ha sido aprobada por la dirección o el rectorado.  Este dato es muy importante ya que sirve para que el sistema pueda asignar en forma automática una Tabla de Precios de Servicios al estudiante.
5 Observaciones Es una nota o comentario adicional sobre la Matrícula del Estudiante.

Con esta información, el Sistema puede crear un nuevo registro de Matrícula para el estudiante.

Ejecución

Para ejecutar el Proceso de Matriculación, sigua los siguientes pasos:

1 Iniciar Asistente de Matrículas Para Iniciar el Asistente de Matrículas, seleccione la opción Matrículas del Módulo Académico del Menú de Inicio de Dobra.   Si no sabe usar el Menú de Inicio de Dobra, consulte el tema de Interface.
2 Código del Estudiante Para Abrir un nuevo registro de Matrícula, es necesario proporcionar el año lectivo, el periodo y el código del estudiante.  Si el código del estudiante es correcto y si no tiene ya un registro de matrícula creado, entonces se activan los controles para ingresar los datos de la Matrícula.
Si los datos no son correctos o el estudiante ya tiene un código de matrícula creado, se presentará un mensaje correspondiente y no se abrirá ningún registro de matrícula.
Para el caso de estudiantes nuevos, use el Botón de Comandos "Nuevo >>" para acceder al formulario de ingreso de nuevos estudiantes.
De igual forma, si se desea actualizar los datos del estudiante, presione el Botón de Comandos "Actualizar datos >>".
3 Asignar Nivel Orgánico Esta información se puede asignar desde la lista jerárquica "Nivel Orgánico".
Para asignar un AREA - JORNADA - CICLO - CARRERA y CURSO, simplemente seleccione con el ratón el curso correspondiente expandiendo el Arbol de Carpetas si es nescesario.
