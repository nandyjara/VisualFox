Módulo de Ventas
Informe: Comisiones

Definición

El Informe de Comisiones le permite al usuario extraer un reporte de todas las liquidaciones de comisiones a los vendedores de la Empresa.

Acceso

Para acceder a esta opción, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el Submenú Ventas y desde la sección Informe seleccione el comando Comisiones.

Elementos

Elemento

Descripción
Buscar Sirve para buscar el documento por número.
Estado El cuadro estado es de "solo lectura" y muestra información acerca del estado actual del documento Ej: ANULADO, APROBADO, ETC.
Vendedor

El cuadro de Vendedor sirve para seleccionar el vendedor quién efectúa la orden de venta.
Términos

El cuadro de Términos sirve para seleccionar el plazo 15, 20, 45 días en que se realiza la orden de venta.
Detalle de Productos y Servicios

La sección de detalle de Productos y Servicios está compuesta por los cuadros código, nombre, cantidad, empaque, total, lista, precio, desc%, desc, extendido, imp.; Estos cuadros son editables y sirven para ingresar los datos de los productos. Además cuenta con el botón de comando nuevo y quitar el cual sirve para quitar o agragar una nueva fila en la cuadrícula, además del cuadro Pares.
Nota que se Imprime en la Orden

Este cuadro es "editable" y sirve para ingresar una descripción impresa de la transacción.
Comentario Interno

Este cuadro es "editable" y sirve para ingresar una comentario  de la transacción para uso interno.
