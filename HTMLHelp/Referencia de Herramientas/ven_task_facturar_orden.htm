Módulo de Ventas
Proceso: Facturar Orden

Definición

El Proceso Facturar Orden sirve para registrar un Egreso de Inventario los productos que han salido por concepto de una Orden de Venta.

Acceso

Para acceder a esta opción, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el Submenú Ventas y desde la sección Proceso seleccione el comando Facturar Orden.

Elementos

Elemento

Descripción
Orden de Venta El cuadro orden de venta sirve para ingresar el numero de la orden de venta.
Tipo El cuadro Tipo es de "solo lectura" y muestra el Tipo de transacción.
Fecha El cuadro Fecha es de "solo lectura" y muestra fecha de la transacción.
Divisas El cuadro divisa es de "solo lectura" sirve para seleccionar la divisa ó se visualiza la divisa según el tipo de cambio.
Cambio El cuadro Cambio es de "sólo lectura" y sirve para visualizar el tipo de cambio.
Orden de compra El cuadro orden de compra sirve para ingresar el numero de la orden de compra.
Fecha El cuadro Fecha es de "solo lectura" y muestra fecha de la transacción.
No. Factura El cuadro de factura del proveedor sirve para ingresar el numero de factura del proveedor.
Cliente El cuadro de factura del proveedor sirve para ingresar el numero de factura del proveedor.
Vendedor El cuadro vendedor es de "solo lectura" sirve para seleccionar el proveedor a quién se le factura la orden.
Término El cuadro Término es de "solo lectura" y sirve para seleccionar la forma de pago pueden ser  a 5, 15, 20 días.
Transporte El cuadro de transporte es "Editable" sirve para seleccionar el medio de transporte de ventas.
Nota que se Imprime en la Orden Este cuadro es "editable" y sirve para ingresar una descripción impresa de la transacción.
Comentario para Uso de la Empresa Este cuadro es "editable" y sirve para ingresar un comentario de la empresa acerca de la transacción.

Crear

Para crear un nuevo proceso, haga lo siguiente:

  1. El cursor se ubica en el cuadro Orden de venta, a continuación ingrese el número de la orden de venta ya registrada.
  2. Luego ingrese el número de Orden de Compra, Fecha que puede ser editable si desea ingresar una nueva fecha respectiva a la orden de compra, No. Factura O/C.

  3. Luego seleccione el transporte a traves del cual se envía la orden de venta.

  4. Luego se visualiza una lista de los productos pedidos en la orden de venta.  Se puede editar solo la columna de Cantidad, que no puede ser mayor al ingreso, caso contrario el sistema genera un error de ingreso de datos. En la columna Order se encuentra la cantidad de mercadería ingresada o recibida ya en bodega; este valor es automáticamente generado por DOBRA.

  5. Seleccione el comando Guardar desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.

  6. A continuación el sistema le mostrará con el nuevo número de documento asignado.
