Módulo de Proveeduria
Documento: Reintegros

Definición

El documento Reintegro de Suministros sirve para elaborar un Informe de todos los suministros que por alguna razón justificada son devueltas a la bodega.

Acceso

Para acceder a esta opción, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el Submenú Proveduria y desde la sección Documento seleccione el comando Reintegros.

Elementos

 

Elemento

Descripción
Buscar Cuadro de Búsqueda por Número, se usa para localizar documentos. Este cuadro se desactiva cuando el formulario entra en modo de edición.
Estado El cuadro estado es de "solo lectura" y muestra información acerca del estado actual del documento Ej: ANULADO, APROBADO, ETC.
ID El cuadro ID es de "solo lectura" y muestra el número de identificación relacional del documento que es generado por el sistema.
Número El cuadro Numero es de "solo lectura" y muestra número secuencial del documento que es generado por el sistema.
Asiento El cuadro Asiento es de "solo lectura" y muestra el número del asiento contable asociado al documento y que es generado por el sistema.
Tipo El cuadro Tipo es de "solo lectura" y muestra el Tipo de transacción.
Fecha El cuadro Fecha es de "editable" y muestra fecha de la transacción.
Divisas El cuadro de divisa sirve para seleccionar la divisa ó se visualiza la divisa según el tipo de cambio.
Concepto

Breve descripción de la Transacción.
Cambio

El cuadro Cambio es de "Sólo lectura" y sirve para visualizar el tipo de cambio.

División

El cuadro de División sirve para seleccionar la división o línea de negocio a la que pertenece el documento
Detalle de reintegro

La sección de Detalle de reintegro está compuesta por los cuadros código, nombre, cantidad, dev., bodega, empaque, total, total dev, costo, extendido; Estos cuadros son editables y sirven para vidualizar todos los productos e ingresar los productos devueltos.
Nota que se Imprime en la Orden

Este cuadro es "editable" y sirve para ingresar una descripción impresa de la transacción.
