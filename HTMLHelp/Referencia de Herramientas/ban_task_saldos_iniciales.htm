Módulo: Caja y Bancos
Proceso: Saldos Iniciales

Definición

Este Asistente sirve para establecer el Saldo inicial de cada una de las Cuentas de Caja y Bancos de las Clases: Banco y Caja de Efectivo. Este proceso no afecta las cuentas contables solo afecta el módulo, por eso con los datos de este Proceso se debe ingresar manualmente por primera vez los valores de Saldos iniciales en el Asiento de Balance de Situación Inicial.

Acceso

Para acceder a esta opción, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el Submenú Caja y Bancos y desde la sección Proceso seleccione el comando Saldos Iniciales.

Elementos

Elemento

Descripción
Fecha El cuadro fecha, sirve para ingresar la fecha del proceso de saldos iniciales.
División El cuadro división sirve para ingresar la división de la empresa.
Cuadrícula de proceso de saldos La cuadrícula del proceso de saldos está compuesta por las columnas de código, nombre, clase, división, cambio, saldo inicial, saldo base.  La cuadrícula del proceso de saldos sirve para ingresar los saldos iniciales de las cuentas en efectivo, caja y bancos.

Comentario

Este Asistente debe ser usado únicamente al iniciar las operaciones con DOBRA® , antes de registrar el Asiento de Situación Inicial Contable.
