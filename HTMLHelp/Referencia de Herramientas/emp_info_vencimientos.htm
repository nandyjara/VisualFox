Módulo: Empleados
Informe: Vencimientos

Definición

Este informe presenta un listado de todos los empleados que tienen deudas vencidas, muestra detalladamente el concepto de la deuda, fecha, valores y saldo por pagar.

Recuerde que cada vez que seleccione algún filtro debe presionar el botón Actualizar ó la tecla F5.

Acceso

Para acceder a esta opción, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el Submenú Empleados y desde la sección Informe seleccione el comando Vencimientos.

Elementos

El Informe de Vencimiento proporciona los siguientes Filtros:

Elemento

Descripción
Fecha de Inicio

Aquí el usuario selecciona la fecha de fin de donde se va a calcular los datos para las formulas.
Fecha de Fin

Aquí el usuario selecciona la fecha de fin de donde se va a calcular los datos para las formulas.
Tipo seleccione el tipo de empleado.
Empleado

Seleccione el nombre de empleado .
