Modulo: Presupuesto
Informe: Presupuestos

Definición

El Informe de Presupuesto muestra el Presupuesto seleccionado.  Este informe selecciona la estructura presupuestaria e interpreta cada una de los códigos presupuestarios.  El Informe de Presupuesto pivota los datos en forma mensual.
El Informe de Presupuesto puede ser INFORMATIVO o COMPARATIVO, además puede interpretar variantes como la aplicación del formato de FLUJO DE CAJA que muestra los saldos iniciales y finales en la parte superior e inferior de cada mes presupuestado.

Acceso

Para acceder a esta opción, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el Submenú Presupuesto y desde la sección Informe seleccione el comando Presupuestos.

Elementos

El Informe de Presupuestos proporciona los siguientes Filtros:

Elemento

Descripción
Tipo de Flujo

El calendario inicial aparece para especificar rangos de fechas.
Aquí se requiere la fecha inicio de la transacción en el informe.
Resumen

Aquí se requiere la fecha del informe.
Comparar con

Se selecciona comparar con, si el presupuesto se compara con real contable (balance general), real flujo de caja(saldos calculados mensualmente) de caja o con nada.
Pivotar Saldos

Sirve para seleccionar la opción de presentar o no, en forma dinámica las columnas de saldos presupuestarios.

RECOMENDACIONES

Generalmente el Informe de Presupuesto comparativo toma más tiempo debido a que DOBRA® realiza el cálculo del importe de las cuentas relacionadas con cada código presupuestario.  El Informe de Presupuesto Comparativo en modo de Flujo de Caja toma mucho más tiempo ya que calcula adicionalmente los saldos que se pivotan mensualmente.
Si el usuario ha cometido un error en una fórmula de algún código presupuestario, DOBRA® mostrará un mensaje de error y el Informe puede presentar resultados incorrectos.

Debido a que el Informe de Presupuesto puede ser pivotado de varias formas, no existe un formato de informe fijo en muchos casos el usuario podría obtener mejores resultados si exporta el Informe a Microsoft® EXCEL®.
