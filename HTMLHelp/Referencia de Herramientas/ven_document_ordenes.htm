Definición

La Orden de Venta sirve para registrar los productos pedidos por el Cliente. Una vez realizada una Orden de Venta en el momento de grabar el documento se abre una pantalla de Destino Administrativo del Documento en donde el usuario debe seleccionar el buzón a donde va a dirigirse la Orden de Venta para ser aprobada, mientras la Orden de Venta no sea aprobada no podrá ser ingresada en el Proceso de Facturar Orden.

La Orden de Venta no registra ningún movimiento contable ni actualiza los saldos de inventario u otros módulos.

 




Integración

No se puede hacer ningun tipo de transacción.

Acceso

Para acceder a esta opción, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el Submenú Ventas y desde la sección Documento seleccione el comando Orden de Venta.

Elementos

Elemento

Descripción
Buscar Sirve para buscar el documento por número.
Estado El cuadro estado es de "solo lectura" y muestra información acerca del estado actual del documento Ej: ANULADO, APROBADO, ETC.
ID El cuadro ID es de "solo lectura" y muestra el número de identificación relacional del documento que es generado por el sistema.
Número El cuadro Numero es de "solo lectura" y muestra número secuencial del documento que es generado por el sistema.
Asiento El cuadro Asiento es de "solo lectura" y muestra el número del asiento contable asociado al documento y que es generado por el sistema.
Tipo El cuadro Tipo es de "solo lectura" y muestra el Tipo de transacción.
Fecha El cuadro Fecha es de "editable" y muestra fecha de la transacción.
Divisa El cuadro de divisa sirve para seleccionar la divisa ó se visualiza la divisa según el tipo de cambio.
Cambio

El cuadro Cambio es de "Sólo lectura" y sirve para visualizar el tipo de cambio.
División

El cuadro de División sirve para seleccionar la división o línea de negocio a la que pertenece el documento
Cliente

El cuadro de Cliente sirve para seleccionar el cliente a quién se le hace la orden de venta.
Vendedor

El cuadro de Vendedor sirve para seleccionar el vendedor quién efectúa la orden de venta.
Términos

El cuadro de Términos sirve para seleccionar el plazo 15, 20, 45 días en que se realiza la orden de venta.
Detalle de Productos y Servicios

La sección de detalle de Productos y Servicios está compuesta por los cuadros código, nombre, cantidad, empaque, total, lista, precio, desc%, desc, extendido, imp.; Estos cuadros son editables y sirven para ingresar los datos de los productos. Además cuenta con el botón de comando nuevo y quitar el cual sirve para quitar o agragar una nueva fila en la cuadrícula, además del cuadro Pares.
Nota que se Imprime en la Orden

Este cuadro es "editable" y sirve para ingresar una descripción impresa de la transacción.
Comentario Interno

Este cuadro es "editable" y sirve para ingresar una comentario  de la transacción para uso interno. 

Crear

Para crear un nuevo Documento, haga lo siguiente:

  1. Seleccione el comando Nuevo desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.

  2. El cursor se ubica en el cuadro división.  Seleccione la división a la que corresponde la factura.  Puede usar las teclas ALT+FLECHA_ABAJO o CTRL+FLECHA_ABAJO.  Acontinuación presione la tecla TAB para mover el cursor al cuadro cliente.

  3. Escriba el nombre del cliente y luego presione la tecla TAB para mover el el cursor al cuadro vendedor.

  4. El cursor se ubica en el cuadro vendedor.  Escriba el nombre del vendedor y luego presione la tecla TAB para mover el el cursor al cuadro término.

  5. Seleccione el témino de pago al que corresponde la orden.  Puede usar las teclas ALT+FLECHA_ABAJO o CTRL+FLECHA_ABAJO.  Acontinuación presione la tecla TAB para mover el cursor al cuadro No. de guía.

  6. Para registrar los productos que serán descritos en la orden, se debe agregar nuevas filas en la cuadrícula de Detalle de Productos.  Para agregar una nueva fila haga un click en el botón () de la sección Detalle de Productos y servicios o presione la tecla ALT+N.  Aparece una nueva fila en la cuadrícula y el cursor se ubica en la primera celda de la fila.

  7. Seleccione el código del producto.  Use la tecla ALT+FLECHA_ABAJO y seleccione el producto.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.  

  8. Haga click en () y aparecerá una ventana de diálogo en la cual podrá escribir una explicación más detallada acerca del producto y presione la tecla TAB para mover el cursor a la siguiente celda. 

     

  9. Ingrese la cantidad del producto.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.

  10. Seleccione el empaque, éste viene dado internamente según el producto de venta.  Use la tecla ALT+FLECHA_ABAJO para seleccionar.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.  

  11. Ingrese el precio del producto.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.

  12. Ingrese el valor del descuento del producto.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.

  13. Luego de haber seleccionado todos los productos para la orden, ud. Puede asignarle el valor del impuesto(IVA) a cada producto.  Haga click en () y el sistema se encargará de asignarle internamente el valor correspondiente del IVA a cada producto, para su calculo posterior. 

  14. Si desea ingresar una nuevo Detalle de Productos y Servicios regrese al paso 6.  Si desea quitar una fila de la cuadrícula de productos y servocios, haga un click en la fila correspondiente y luego haga un click el botón () de la sección Detalle de Productos y servicios ó presione la tecla ALT+Q.  

  15. Escriba la Nota que va a ser impresa en la orden y además de un comentario interno para uso exclusivo de la empresa. 

  16. Seleccione el comando Guardar desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.

  17. Acontinuación el sistema le mostrará con el nuevo número de documento asignado.
