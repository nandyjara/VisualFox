Módulo de Seguridad
Mantenimiento: Usuarios

Definición

El Fichero de Usuarios facilita la creación de nuevos usuarios, así como la edición o anulación de usuarios existentes. Además, muestra el Perfil de Seguridad del Usuario.

Acceso

Para acceder a esta opción, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el Submenú Seguridad y desde la sección Mantenimiento seleccione el comando Usuarios.

Elementos

Elemento

Descripción
Buscar Sirve para buscar el documento por número.
ID El cuadro ID es de "solo lectura" y muestra el número de identificación relacional del documento que es generado por el sistema.
Código El cuadro código, sirve para ingresar el código del usuario.
Contraseña El cuadro contraseña, sirve para ingresar la clave del usuario para poder tener acceso al sistema.
Confirmar El cuadro confirmar, sirve para ingresar nuevamente la clave del usuario para tener acceso al sistema.
Nombre Este cuadro sirve para ing4resar el nombre del usuario.
Grupos El cuadro grupos, sirve para seleccionar el grupo al que pertenece el usuario.
Nota

Este cuadro es "editable" y sirve para ingresar un comentario acerca sel usuario.
Estado

El cuadro estado, sirve para especificar el estado del usuario si está activo o inactivo.
Perfil de Acceso

La estructura jerárquica perfil de acceso está compuesta por los botones de comando permitir todo, negar todo y copiar de.  El árbol de jerarquía sirve para permitir y especificar el acceso de los usuarios  a los diferentes módulo del sistema.

Crear

Para crear una nuevo fichero de usuarios, haga lo siguiente:

  1. Seleccione el comando Nuevo desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.

  2. El cursor se ubica en el cuadro, sirve para ingresar código y nombre del usuario.
  3. Ingrese la contraseña y la confirmación del usuario para permitirle el respectivo acceso.
  4. Ingrese el grupo al que pertenece el usuario.

  5. Luego puede escribir una glosa con el comertario acerca del usuario. 

  6. Defina el estado del usuario, si está activo o inactivo.

  7. La estructura jerárquica perfil de acceso está compuesta por los botones de comando permitir todo, negar todo y copiar de.  El árbol de jerarquía sirve para especificar el acceso de los usuarios  a los diferentes módulo del sistema. Si desea permitirle el acceso a todos los módulos del sistema haga click en el botón de comandos ().  Si no desea permitir el acceso a ninguno de los módulos del sistema al usuario, haga click en el botón del comandos (). 

  8. Seleccione el comando Guardar desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.

  9. Acontinuación el sistema le mostrará con el nuevo número de documento asignado.

Modificar

Para modificar un Fichero de usuarios, haga lo siguiente:

  1. Seleccione el comando Modificar desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.
  2. A continuación se pueden editar todos los campos del fichero usuarios, para mas detalles del ingreso de cada uno de los datos regrese a los pasos del procedimiento crear.

Eliminar

Para eliminar un Fichero de usuarios, haga lo siguiente:

  1. Seleccione el comando Eliminar desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.

  2. A continuación, para eliminar el registro completamente haga click en el botón si de la ventana que aparece o click en no; si no esta seguro de eliminar el registro.
