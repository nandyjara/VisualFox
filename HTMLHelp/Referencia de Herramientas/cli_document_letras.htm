Módulo: Clientes
Documento: Letras de Cambio

Definición

El documento de Letras de Cambio sirve para reemplazar las deudas pendientes de un cliente con letras de cambio.  Adicionalmente, este documento permite mantener un registro de cheques postfechados entregados como garantía a las letras de cambio.

El Documento de Letras de Cambio  no representa un Ingreso de Efectivo para la Empresa, por esta razón, no se genera un Comprobante de Ingreso, pero sí determina el reemplazo de una parte o toda una deuda por un conjunto de Letras de Cambio.

Note que las Letras de Cambio  reemplazan las deudas que fueron aplicadas; por ejemplo, si un cliente adeuda una Factura de Venta y registra letras de cambio que abonan la factura, el Saldo del Cliente no varía; es decir, el Cliente sigue adeudando la misma cantidad de dinero pero representado en Letras de Cambio, ya no debe una Factura de Venta sino que ahora debe un conjunto de Letras de Cambio.


Integración

Acceso

Para acceder a esta opción, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el Submenú Clientes y desde la sección Documento seleccione el comando Letras de Cambio.

Elementos

Elemento

Descripción
Buscar Sirve para buscar el documento por número.
Estado El cuadro estado es de "solo lectura" y muestra información acerca del estado actual del documento Ej: ANULADO, APROBADO, ETC.
ID El cuadro ID es de "solo lectura" y muestra el número de identificación relacional del documento que es generado por el sistema.
Número El cuadro Numero es de "solo lectura" y muestra número secuencial del documento que es generado por el sistema.
Asiento El cuadro Asiento es de "solo lectura" y muestra el número del asiento contable asociado al documento y que es generado por el sistema.
Tipo El cuadro Tipo es de "solo lectura" y muestra el Tipo de transacción.
Fecha El cuadro Fecha es "editable" y muestra fecha de la transacción.
División El cuadro de lista de división sirve para seleccionar la división o línea de negocio a la que pertenece el documento
Concepto

El cuadro Concepto es "editable" y sirve para ingresar una descripción corta o abreviada del motivo de la transacción.
Cliente

El cuadro de Cliente sirve para seleccionar el nombre del cliente quien adeuda las letras de cambio.
Detalle de las Letras

La sección de Detalle de las Letras está compuesto por una cuadrícula de Letras que tiene las columnas de número de letras, tipo, vencimiento, detalle, divisa, valor, valor base y además de los botones de comandos Nuevo y Quitar que sirve para agregar o quitar filas en la cuadrícula y el incrementador de N0. cuotas.  La sección de detalle de las Letras sirve para registrar el número ó cantidad de letras que el cliente debe a la empresa como forma de pago.
La sección de Detalle de las Letras cuenta además con el total de Letras.

Detalle de  Duedas del Cliente

La sección de Detalle de Deudas del Cliente está compuesta por una cuadrícula de cuentas contables que tienen las siguientes columnas: emisión, vence, tipo, número, concepto, divisa, saldo, abono, nuevo saldo.
La sección de detalle de Deudas del Cliente sirve para visualizar automáticamente las mismas. 
La sección de Detalle de Deudas del Cliente cuenta además con el total deudas, total crédito, total abonos, nuevo saldo, déficit de ingresos , superávit.

Observaciones

El cuadro observaciones es "editable" y sirve para ingresar una descripción completa de la transacción.

Crear

Para crear una nueva Cuenta, haga lo siguiente:

  1. Seleccione el comando Nuevo desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.

  2. El cursor se ubica en el cuadro concepto.  Escriba el concepto o motivo de la transacción y luego presione la tecla TAB para mover el el cursor al cuadro cliente.

  3. Seleccione el cliente.  Puede usar las teclas ALT+FLECHA_ABAJO o CTRL+FLECHA_ABAJO.  A continuación presione la tecla tab para mover el cursor al cuadro No. Cuotas.

  4. Haga click en el spiner (incrementador) ( ).  A continuación presione la tecla tab para mover el cursor la sección de Detalle de las Letras.

  5. Para registrar las letras efectuadas debe agregar nuevas filas en la cuadrícula.  Para agregar una nueva fila haga un click en el botón () de la sección de letras o presione la tlecla ALT+N.  Aparece una nueva fila en la cuadrícula y el cursor se ubica en la primera celda de la fila.

  6. Seleccione el tipo de pago que puede ser en letras o pagaré previamente el sistema genera automáticamente el número de letra de cambio o pagaré.  Puede usar las teclas ALT+FLECHA_ABAJO o CTRL+FLECHA_ABAJO.   A continuación y presione la tecla TAB para mover el cursor a la siguiente celda.

  7. Escriba la fecha de vencimiento de pago y luego presione la tecla TAB para mover el el cursor a la siguiente celda.

  8. Escriba el detalle de la letra de cambio ó pagaré y luego presione la tecla TAB para mover el el cursor a la siguiente celda.

  9. Seleccione la divisa del pago.  Generalmente en la celda de divisa aparece seleccionada la divisa base del sistema, si no necesita cambiar la divisa entonces solo presione la tecla TAB para mover el cursor a la siguiente celda; en caso contrario, use la tecla ALT+FLECHA_ABAJO y seleccione la divisa.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.  

  10. Escriba el valor según la divisa seleccionada y luego presione la tecla TAB para mover el el cursor a la siguiente celda.

  11. Si desea ingresar un nuevo tipo de pago regrese al paso 5.  Si desea quitar una fila de la cuadrícula de letras, haga un click en la fila correspondiente y luego haga un click el botón () de la sección Detalle de las Letras o presione la tecla ALT+Q.  

  12. Presione CTRL+TAB para mover el cursor a la sección detalle de deudas del cliente.  Se visualizará automáticamente las deudas pendientes de los clientes.  Ud. puede escribir o repartir los valores de los abonos del cliente según el valor de Total Letras.

  13. Luego puede escribir una observación con el detalle de letras de cambio.  Cuando este listo debe asegurarse que todos los valores totales estén correctamente calculados.

    Seleccione el comando Guardar desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.

  14. A continuación el sistema le mostrará con el nuevo número de documento asignado.
