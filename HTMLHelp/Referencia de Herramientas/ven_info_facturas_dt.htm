Módulo de Ventas
Informe: Productos Facturados

Definición

Este informe presenta un listado de todos los Productos facturados por varios filtros como son por periodo de tiempo .

Acceso

Para acceder a esta opción, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el Submenú Ventas y desde la sección Informe seleccione el comando Productos Facturados.

Elementos

 

Elemento

Descripción
Buscar Sirve para buscar el documento por número.
Comentario Interno

Este cuadro es "editable" y sirve para ingresar una comentario  de la transacción para uso interno.
