Módulo: Ventas
Introducción

El Módulo de Ventas proporciona herramientas para el registro de las Facturas de Venta que la empresa hace a sus clientes.  Aunque el usuario puede registrar facturas de venta directas en un solo paso, DOBRA®  proporciona un mecanismo basado en una Orden de Venta.

Venta Directa

Este Método de Venta se usa principalmente para registrar ventas de Contado en donde la información de crédito o los procesos de Aprobación de créditos no son necesarios.

Venta mediante Orden de Venta

En este método, el cliente hace un Pedido u Orden de Venta que es registrada en el Sistema.  Esta Orden de Venta tiene que pasar por un proceso de Aprobación basado en el Subsistema de Correo de DOBRA® , y finalmente, esta Orden de Venta aprobada pasa a ser facturada.

Facturación de Mercaderías  y Servicios

DOBRA®  tiene la capacidad de facturar Mercaderías  y Servicios .  Estos dos tipos de productos se diferencian en que las Mercaderías tienen un costo de venta y una existencia para facturar; los servicios, por otro lado, representan Ingresos para la empresa, no tienen costo o su costo no se determina en el momento de la venta y no se pueden cuantificar por lo que no es necesario que tengan existencias o saldos.

Para el usuario esto debe ser transparente, lo único que notará al momento de registrar Documentos de Venta es que cuando se trata de un servicio no se requiere seleccionar una Bodega para facturarlo.

Impresión de Facturas

DOBRA®  imprime de forma predeterminada un Comprobante de Venta cuando se registra una Factura de Venta; pero, si el usuario necesita que DOBRA®  imprima la Factura en un Formato Pre-Impreso , debe realizar 3 pasos:

Instalar (aunque ya exista) una nueva impresora en Windows® y establecer un tamaño definido por el usuario.  Note que esta nueva impresora no debe ser la impresora predeterminada.

Crear un Archivo de Formato que use el tamaño definido por el usuario de la nueva impresora y que cuadre con el tamaño del Formato Pre-Impreso  de la empresa. Este Archivo de Formato debe ser guardado en el directorio predeterminado de reportes.  Para más información sobre cómo crear un Formato para un reporte, consulte el Capítulo de Reportes Agregados.

A continuación, el usuario debe editar el parámetro del sistema: FORMATO-FACTURA-VENTA y asignarle como valor el nombre del nuevo archivo de formato.
