Módulo: Compras
Documento:  Facturas

Definición

La Factura de Orden de Compra sirve para registrar la Factura de Compra emitida por el Proveedor por los productos que ya fueron recibidos en la empresa.

Integración

Acceso

Para acceder a esta opción, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el Submenú compras y desde la sección Documento seleccione el comando Facturas.

Elementos

Elemento

Descripción
Buscar Sirve para buscar el documento por número.
Estado El cuadro estado es de "solo lectura" y muestra información acerca del estado actual del documento Ej: ANULADO, APROBADO, ETC.
ID El cuadro ID es de "solo lectura" y muestra el número de identificación relacional del documento que es generado por el sistema.
Número El cuadro Numero es de "solo lectura" y muestra número secuencial del documento que es generado por el sistema.
Asiento El cuadro Asiento es de "solo lectura" y muestra el número del asiento contable asociado al documento y que es generado por el sistema.
Tipo El cuadro Tipo es de "solo lectura" y muestra el Tipo de transacción.
Fecha El cuadro Fecha es "editable" y muestra fecha de la transacción.
Divisa El cuadro de divisa sirve para seleccionar la divisa ó visualizar ésta divisa según el tipo de cambio.
Cambio El cuadro Cambio es de "Sólo lectura" y sirve para visualizar el tipo de cambio.
Proveedor El cuadro de Proveedor sirve para seleccionar el código y nombre del proveedor.
Fact. Prov. El cuadro factura de provedor sirve para ingresar el número de factura del proveedor.
Término El cuadro de Término sirve para seleccionar el plazo 15, 20, 45 días, contado en que se realiza la factura.
División El cuadro de División sirve para seleccionar la división o línea de negocio a la que pertenece el documento.
O. Compra El cuadro de O. Compra  es de "solo lectura" y muestra la orden de compra que es generado por el sistema.
Detalle de Productos La sección de Detalle de Productos está compuesto por una cuadrícula de productos que tiene las columnas de código, nombre, bodega, cantidad, empaque, unidades, precio, desc%, desc, total y además de los botones de comandos Nuevo y Quitar que sirve para agregar o quitar filas en la cuadrícula. La sección de detalle de Productos sirve para registrar la cantidad de productos que van a ser facturados además contiene el subtotal, descuento, impuestos, total.
Nota que se Imprime en la Factura El cuadro observaciones es "editable" y sirve para ingresar una descripción completa de la transacción.

Crear

Para crear una nuevo Documento, haga lo siguiente:

  1. Seleccione el comando Nuevo desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.

  2. El cursor se ubica en el cuadro Proveedor.  Escriba el nombre del proveedor y luego presione la tecla TAB para mover el el cursor al cuadro Fact.Prov.

  3. Ingrese la factura del proveedor.  Acontinuación presione la tecla TAB para mover el cursor al cuadro término.

  4. Seleccione el témino de pago al que corresponde la factura.  Puede usar las teclas ALT+FLECHA_ABAJO o CTRL+FLECHA_ABAJO.  Acontinuación presione la tecla TAB para mover el cursor al cuadro división.

  5. Seleccione la división a la que corresponde la factura.  Puede usar las teclas ALT+FLECHA_ABAJO o CTRL+FLECHA_ABAJO.  Acontinuación presione la tecla TAB para mover a la sección de Detalle de Productos.

  6. Para registrar los productos que serán descritos en la factura, se debe agregar nuevas filas en la cuadrícula de Detalle de Productos.  Para agregar una nueva fila haga un click en el botón () de la sección Detalle de Productos o presione la tecla ALT+N.  Aparece una nueva fila en la cuadrícula y el cursor se ubica en la primera celda de la fila.

  7. Haga click en  () y aparecerá una ventana de dialogo en la cual se visualizará automáticamente toda la información que esta en el archivo de empleados que sirven para llevar un mejor control. El usuario tiene la opción de modificar los valores de los rubros de egreso para que no sean cancelados en el rol actual si no en el siguiente rol de pago.  acontinuación presione el botón               () para que se actualice la información que va a ser liquiada en el rol.

  8. Seleccione el código del producto.  Use la tecla ALT+FLECHA_ABAJO y seleccione el producto.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.  

  9. Seleccione la bodega de donde disminuyen los productos.  Use la tecla ALT+FLECHA_ABAJO para seleccionar.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.  

  10. Ingrese la cantidad del producto.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.

  11. Seleccione el empaque, éste viene dado internamente según el producto de venta.  Use la tecla ALT+FLECHA_ABAJO para seleccionar.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.  

  12. Ingrese el valor del descuento del producto.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.

  13. Si desea ingresar una nuevo Detalle de Productos regrese al paso 7.  Si desea quitar una fila de la cuadrícula de productos, haga un click en la fila correspondiente y luego haga un click el botón () de la sección Detalle de Productos ó presione la tecla ALT+Q.  

  14. Luego puede escribir una glosa con el detalle de la factura.  Cuando este listo debe asegurarse que todos los valores estén calculados correctamente.

  15. Seleccione el comando Guardar desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.

  16. Acontinuación el sistema le mostrará con el nuevo número de documento asignado.
