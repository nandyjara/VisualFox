Módulo de Proveeduria
Proceso:Saldos Iniciales

Definición

Este Asistente sirve para establecer el Saldo inicial de cada una de las Cuentas de proveeduria. Este proceso no afecta las cuentas contables solo afecta el módulo, por eso con los datos de este Proceso se debe ingresar manualmente por primera vez los valores de Saldos iniciales.

Comentarios

Este Asistente debe ser usado únicamente al iniciar las operaciones con DOBRA, antes de registrar el Asiento de Situación Inicial Contable.

Acceso

Para acceder a esta opción, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el Submenú Proveduria y desde la sección Proceso seleccione el comando Saldos Iniciales.

Elementos

El Proceso de Saldos Iniciales proporciona los siguientes Filtros:

Elemento

Descripción
Empleado El cuadro empleado sirve para seleccionar el código y nombre del empleado.
Fecha El cuadro fecha sirve para ingresar la fecha del proceso del saldo inicial.
Cuadrícula de Proceso de saldos La cuadrícula del proceso de saldos está compuesto por las columnas de rubro, división, fecha, vence, detalle, divisa, valor y total base.  La cuadrícula del proceso de saldos Lista las deudas iniciales del Empleado. Además dicha cuadrícula cuenta con los botones de comandos nuevo y quitar que sirven para agregar y para quitar una celda de la cuadrícula.

Crear

Para crear una nuevo Proceso, haga lo siguiente:

  1. Seleccione el comando Nuevo desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.
  2. Seleccionar la carpeta, ésta lista es tomada del fichero Grupos módulo Proveeduria, bodega, división para saldos iniciales.

  3. Ingrese los Detalle de Saldos Iniciales como son: código, nombre, stock, divisa, costo, costo base, saldo base.

  4. Seleccione el comando Guardar desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.

  5. A continuación el sistema le mostrará con el nuevo número de documento asignado.
