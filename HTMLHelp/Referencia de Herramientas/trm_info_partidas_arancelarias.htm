Módulo de Trámites
Informe:  Partidas Arancelarias

Definición

El Informe de Partidas Arancelarias le permite al usuario extraer un reporte de cada una de las ordenes con sus respectivas partidas arancelarias.

Integración

No se puede realizar ningún tipo de transacción.

Acceso

Para acceder a esta opción, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el Submenú Trámites y desde la sección Informe seleccione el comando Partidas Arancelarias.

Elementos

Elemento

Descripción
Filtros En la parte superior del informe encontramos varios cuadros para filtrar la información de partidas como: Fechas, Partida Arancelaria, Pais, Importador, Año, Orden. 
Cuadrícula de información El informe presenta datos de: Partida, Orden, País, Importador, Cantidad (Uni), Costo.

Crear

Para crear una nuevo Informe, haga lo siguiente:

  1. Seleccionar el rango de fechas o cualquiera de los otros criterios que desea consultar y presione la tecla F5 o el botón consultar.

  2. Acontinuación el sistema le mostrará los datos de acuerdo a los criterios de busqueda seleccionados por el usuario.
