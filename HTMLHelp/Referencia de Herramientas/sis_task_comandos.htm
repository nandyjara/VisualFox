Módulo de Sistemas
Proceso: Comandos

Definición

La Ventana de Programación sirve para que el usuario pueda crear programas basados en secuencias de comandos que operan sobre la base de datos.

La ventana de Programación le da muchas posibilidades al usuario para realizar procesos que no están incluidos con el sistema.
Los programas que el usuario escribe en el cuadro de texto pueden ser almacenados como archivos de comandos que llevan la extensión DBS.  Para esto, cuando la ventana de programación está activa, el usuario puede usar los botones de comando: Nuevo, Abrir y Guardar de la Barra de Herramientas.

 

DOBRA® le da la posibilidad al usuario de usar cualquier comando disponible en tiempo de ejecución del lenguaje de programación Microsoft® Visual FoxPro®.  Para más información sobre este lenguaje puede visitar el siguiente sitio en Internet:

www.msdn.microsoft.com/library/devprods/vs6/vfoxpro/foxhelp

El usuario puede usar asteriscos al inicio de cada línea de programa para ingresar comentarios.
DOBRA® sólo interpreta secuencias de comandos, no se deben usar estructuras de control de flujo de programa como IF...ELSE...ENDIF | DO CASE... | DO WHILE... | FOR...ENDFOR | SCAN...ENDSCAN, etc.
El usuario puede usar cualquier tipo de sentencia, inclusive sentencias que dañen físicamente los datos o que destruyan la información por esta razón, el uso de la Ventana de Programación debe estar autorizado a personal capacitado.
Por otro lado, el laboratorio eventualmente puede solicitar al usuario a que use esta ventana para realizar procesos de Asistencia Técnica telefónica.


Acceso

Para acceder a esta opción, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el Submenú Sistemas y desde la sección Proceso seleccione el comando Comandos.

Elementos

 

Elemento

Descripción
Nuevo El botón de comandos nuevo, sirve para abrir una nueva ventana en blanco, para escribir nuevas líneas de programación.
Abrir El botón de comandos abrir, sirve para abrir una nueva ventana de programación.
Guardar El botón de comandos guardar, sirve para guardar o grabar las líneas de programación escritas, éste archivo es guardado con extensión DBS.
Fuente El botón de comandos fuente, sirve para dar formato a la letra, cambiando el tipo, fuente, tamaño.
Ejecutar El botón de comandos ejecutar, sirve para ejecutar o correr las líneas de programación fuente escritas en el lenguaje de programación Visual Fox.

Crear

Para crear un proceso de comandos, haga lo siguiente:

  1. Haga click sobre el botón de comandos nuevo, y empiece a escribir la secuencia de comandos bajo el lenguaje de programación Microsoft® Visual FoxPro®.

  2. Si desea abrir un archivo ya existente, haga click en el botón de comandos abrir, luego s visualiza una ventana de dialogo abrir, seleccione el archivo deseado y haga click en el botón aceptar, y finalmente se visualiza en la ventana de programación fuente el archivo abierto.

  3. Para guardar nuevas líneas de programación o guardar cambios realizados sobre un archivo, haga click sobre el botón de comandos Guardar y se visualizará una nueva ventana de diálogo donde se debe seleccionar la carpeta dentro de la cual se guardará el archivo y escriba el nombre del archivo en el cuadro Guardar Como.  

  4. Si desea darle formato a la letra, en la ventana de programación haga click en en el botón de comandos Fuente, aparecerá una ventana de dialogo llamada fuente donde puede elegir la fuente ,el estilo de fuente y el tamaño de la letra.

  5. Haga click sobre el botón de comandos ejecutar, el cual sirve para ejecutar o correr las líneas de programación fuente escritas bajo el lenguaje de programación Visual Fox.
