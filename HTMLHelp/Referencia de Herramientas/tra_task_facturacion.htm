Módulo de Transporte
Procesos: Facturación

Definición

El proceso de facturación se utiliza para realizar la facturación a los clientes de todos los servicios y reembolsos involucrados en ordenes de trabajo, este proceso genera una cuenta por cobrar al cliente.

Acceso

Para acceder a esta opción, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el Submenú Transporte y desde la sección Procesos seleccione el comando Facturación.

Elementos

Elemento

Descripción
Cliente Lista desplegable de clientes
Fecha El cuadro Fecha es de "editable" y muestra fecha de la transacción.
Tipo El cuadro Tipo es de "solo lectura" y muestra el Tipo de transacción.
Divisas El cuadro de divisa sirve para seleccionar la divisa ó se visualiza la divisa según el tipo de cambio.
Cambio

El cuadro Cambio es de "Sólo lectura" y sirve para visualizar el tipo de cambio.

Fact. Reembolso Número de factura de reembolso preimpreso
Vendedor Lista desplegable de vendedores
Término Servicios El cuadro de Términos sirve para seleccionar el plazo 15, 20, 45 días en que se realiza la factura de venta de servicios.
Término Reembolsos El cuadro de Términos sirve para seleccionar el plazo 15, 20, 45 días en que se realiza la factura de venta de reembolsos.
División El cuadro de División sirve para seleccionar la división o línea de negocio a la que pertenece el documento
No. factura Número de factura de servicios preimpreso
Pestaña Servicios Esta cuadrícula se llena automáticamente con todos los servicios registrados en ordenes de transporte, el usuario debe marcar las líneas a facturar para que se incluyan en el proceso de facturación
Pestaña Reembolsos Esta cuadrícula se llena automáticamente con todos los reembolsos registrados en ordenes de transporte, el usuario debe marcar las líneas a facturar para que se incluyan en el proceso de facturación
Notas  En la parte inferior del formulario se registra información adicional como: Comentario del cliente, Nota que se imprime en la factura de servicios, Nota que se imprime en la factura de reembolsos

Crear

Para crear un nuevo proceso de facturación haga lo siguiente:

  1. Seleccione el cliente a facturar de la lista desplegable de clientes, puede usar las teclas CTRL+Flecha abajo.
  2. A continuación el sistema llena los datos de las pestañas de Servicios y Reembolsos, el usuario debe ingresar los datos que faltan como: números de facturas preimpresas, vendedor términos de pagos y división.
  3. A continuación debe chequear todos las líneas de servicios y reembolsos que se incluirán en el proceso, y llenar las respectivas notas de comentarios.
  4. Seleccione el comando Guardar desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.
  5. El sistema  limpia los datos para dejar el formulario listo para otra facturación.
