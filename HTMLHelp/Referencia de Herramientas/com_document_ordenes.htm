Módulo: Compras
Documento:  Ordenes

Definición

La Orden de Compra sirve para registrar los productos solicitados al Proveedor.
La Orden de Compra no registra ningún movimiento contable ni actualiza los saldos de inventario u otros módulos.

Acceso

Para acceder a esta opción, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el Submenú compras y desde la sección Documento seleccione el comando Ordenes.

Elementos

Elemento

Descripción
Buscar Sirve para buscar el documento por número.
Estado El cuadro estado es de "solo lectura" y muestra información acerca del estado actual del documento Ej: ANULADO, APROBADO, ETC.
ID El cuadro ID es de "solo lectura" y muestra el número de identificación relacional del documento que es generado por el sistema.
Número El cuadro Numero es de "solo lectura" y muestra número secuencial del documento que es generado por el sistema.
Asiento El cuadro Asiento es de "solo lectura" y muestra el número del asiento contable asociado al documento y que es generado por el sistema.
Tipo El cuadro Tipo es de "solo lectura" y muestra el Tipo de transacción.
Fecha El cuadro Fecha es de "editable" y muestra fecha de la transacción.
Divisas El cuadro de divisa sirve para seleccionar la divisa ó se visualiza la divisa según el tipo de cambio.
Cambio

El cuadro Cambio es de "Sólo lectura" y sirve para visualizar el tipo de cambio.

Proveedor

El cuadro Proveedor sirve para seleccionar el proveedor a quién se le hace la orden de compra.
Términos

El cuadro de Términos sirve para seleccionar el plazo 15, 20, 45 días, de contado, etc; en que se realiza la orden de venta.
División

El cuadro de División sirve para seleccionar la división o línea de negocio a la que pertenece el documento
Detalle de Productos 

La sección de Detalle de Productos está compuesta por los cuadros código, nombre, stock, cantidad, empaque, total, precio, desc%, desc, extendido; Estos cuadros son editables y sirven para ingresar los datos de productos. Además cuenta con el botón de comando nuevo y quitar el cual sirve para quitar o agregar una nueva fila en la cuadrícula.
Nota que se Imprime en la Orden

Este cuadro es "editable" y sirve para ingresar una descripción impresa de la transacción.

Crear

Para crear un nuevo Documento, haga lo siguiente:

  1. Seleccione el comando Nuevo desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.

  2. El cursor se ubica en el cuadro divisas.  Seleccione la divisa correspondiente.  Puede usar las teclas ALT+FLECHA_ABAJO o CTRL+FLECHA_ABAJO.  A continuación presione la tecla TAB para mover el cursor al cuadro proveedor.

  3. Escriba el nombre del proveedor y luego presione la tecla TAB para mover el el cursor al cuadro término.

  4. Seleccione el término de pago correspondiente a la orden.  Puede usar las teclas ALT+FLECHA_ABAJO o CTRL+FLECHA_ABAJO.  A continuación presione la tecla TAB para mover el cursor al cuadro división.

  5. Seleccione la división o línea.  Puede usar las teclas ALT+FLECHA_ABAJO o CTRL+FLECHA_ABAJO.  A continuación presione la tecla TAB para mover el cursor a la sección de Detalle de Productos.

  6. Para registrar los productos que serán descritos en la orden, se debe agregar nuevas filas en la cuadrícula de Detalle de Productos.  Para agregar una nueva fila haga un click en el botón () de la sección Detalle de Productos y servicios o presione la tecla ALT+N.  Aparece una nueva fila en la cuadrícula y el cursor se ubica en la primera celda de la fila.

  7. Seleccione el código del producto.  Use la tecla ALT+FLECHA_ABAJO y seleccione el producto.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.  

  8. Ingrese la cantidad del producto.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.

  9. Seleccione el empaque, éste viene dado internamente según el producto.  Use la tecla ALT+FLECHA_ABAJO para seleccionar.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.  

  10. Ingrese el precio del producto.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.

  11. Ingrese el valor del descuento del producto.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.

  12. Si desea ingresar una nuevo Detalle de Productos  regrese al paso 6.  Si desea quitar una fila de la cuadrícula de productos, haga un click en la fila correspondiente y luego haga un click el botón () de la sección Detalle de Productos ó presione la tecla ALT+Q.  

  13. Escriba la Nota que va a ser impresa en la orden, al finalizar debe asegurarse de que todos los valores totales estén correctamente calculados. 

  14. Seleccione el comando Guardar desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.

  15. A continuación el sistema le mostrará con el nuevo número de documento asignado.
