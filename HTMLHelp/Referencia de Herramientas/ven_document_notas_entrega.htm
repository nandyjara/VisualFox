Módulo de Ventas
Proceso: Nota de Entrega

Definición

Las Notas de Entrega sirve para crear un informe de todos los productos que van a   ser entregados al Cliente.

Acceso

Para acceder a esta opción, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el Submenú Ventas y desde la sección Documento seleccione el comando Nota de Entrega.

Elementos

 

Elemento

Descripción
Buscar Sirve para buscar el documento por número.
Estado El cuadro estado es de "solo lectura" y muestra información acerca del estado actual del documento Ej: ANULADO, APROBADO, ETC.
ID El cuadro ID es de "solo lectura" y muestra el número de identificación relacional del documento que es generado por el sistema.
Número El cuadro Numero es de "solo lectura" y muestra número secuencial del documento que es generado por el sistema.
Tipo El cuadro Tipo es de "solo lectura" y muestra el Tipo de transacción.
Fecha El cuadro Fecha es de "editable" y muestra fecha de la transacción.
División

El cuadro de División sirve para seleccionar la división o línea de negocio a la que pertenece el documento
Transporte El cuadro Transporte es "editable" y sirve para ingresar el tipo de transporte por donde va a ser llevada la mercadería.
Documento El cuadro de divisa sirve para seleccionar la divisa ó se visualiza la divisa según el tipo de cambio.
Referencia

El cuadro Cambio es de "Sólo lectura" y sirve para visualizar el tipo de cambio.

Cliente

El cuadro Proveedor sirve para seleccionar el proveedor a quién se le hace el pedido.
Detalle de una Nota de Entrega

La sección de Detalle de Nota de Entrega está compuesta por los cuadros código, nombre, order, empaque, bodega, cantidad, total, Back-O; Estos cuadros son editables y sirven para ingresar los productos que van junto con la factura de venta.

Crear

Para crear una nuevo Proceso, haga lo siguiente:

  1. Seleccione el comando Nuevo desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.
  2. El cursor se ubica en el cuadro Proveedor.  Escriba el nombre del proveedor y luego presione la tecla TAB para mover el el cursor al cuadro Fact.Prov.
  3. Ingrese la factura del proveedor. Luego presione la tecla TAB para mover el cursor al cuadro término.
  4. Seleccione el término de pago al que corresponde la factura.  Puede usar las teclas ALT+FLECHA_ABAJO o CTRL+FLECHA_ABAJO.  Acontinuación presione la tecla TAB para mover el cursor al cuadro seguro.
  5. Ingrese el transporte, puede ser aéreo, terrestre, marítimo.  Acontinuación presione la tecla TAB para mover el cursor al cuadro peso neto aproximado.
  6. Ingrese el peso neto aproximado de la mercadería.  Acontinuación presione la tecla TAB para mover el cursor a la sección de detalle de pedido.
  7. El cuadro Fecha Embarque es "editable" y muestra fecha en que se hizo el embarque del pedido.
  8. Ingrese el país de origen de donde sale el embarque.  Acontinuación presione la tecla TAB para mover el cursor al cuadro negociación.
  9. El cuadro Puerto de Embarque es "editable" y sirve para ingresar el puerto de donde se embarca la mercadería.
  10. Ingrese el embalaje.  Acontinuación presione la tecla TAB para mover el cursor al cuadro marca y números.
  11. Ingrese la negociación, puede ser FOB (flete a su puerto de destino), CIF (flete a su puerto de destino con seguro incluido).  Acontinuación presione la tecla TAB para mover el cursor al cuadro transporte.
  12. Ingrese el seguro, puede ser propio o del proveedor..  Acontinuación presione la tecla TAB para mover el cursor al cuadro pago.
  13. Ingrese el pago, puede ser giro directo, cheque, tarjeta.  Acontinuación presione la tecla TAB para mover el cursor al cuadro empaque.

  14. Para registrar los productos que serán descritos en el pedido de importación, se debe agregar nuevas filas en la cuadrícula de Detalle de Pedido.  Para agregar una nueva fila haga un click en el botón () de la sección Detalle de Pedido o presione la tecla ALT+N.  Aparece una nueva fila en la cuadrícula y el cursor se ubica en la primera celda de la fila.

  15. Seleccione el código del producto.  Use la tecla ALT+FLECHA_ABAJO y seleccione el producto.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.  

  16. Ingrese peso en Kg. del producto.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.
  17. Ingrese el pedido del producto.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.

  18. Ingrese el precio del producto.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.

  19. Si desea ingresar una nuevo Detalle de Pedido regrese al paso 14.  Si desea quitar una fila de la cuadrícula de pedido, haga un click en la fila correspondiente y luego haga un click el botón () de la sección Detalle de Pedido ó presione la tecla ALT+Q.  

  20. Seleccione el comando Guardar desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.

  21. Acontinuación el sistema le mostrará con el nuevo número de documento asignado.

Modificar

DOBRA® no permite modificar una Nota de Entrega.

Eliminar

DOBRA® permitirá eliminar el documento siempre que, no esté anulado.  Para anular una Nota de Entrega, haga lo siguiente:

  1. Ejecute el comando: Anular, haga un click en: o presione: [CTRL+Del].
  2. DOBRA® muestra un mensaje de confirmación, haga click en el comando SI.
  3. DOBRA® muestra un Cuadro de Diálogo para escribir el motivo de la anulación.  Este campo es obligatorio para poder anular y el texto será agregado a la Nota del documento. 
  4. Escriba el motivo de la anulación y haga click en el comando Aceptar.
  5. Cuando se anula una Nota de Entrega, se realiza lo siguiente:
    La Nota de Entrega se marca como anulado para evitar que el usuario lo vuelva a anular accidentalmente.
