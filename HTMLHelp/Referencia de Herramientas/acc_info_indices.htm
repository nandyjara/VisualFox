Módulo de Contabilidad
Informe de Indices Financieros

Definición

Este es un informe interactivo que le permite al usuario establecer formulas para calcular una gran variedad de índices financieros Ejem: Razón Corriente, Prueba Acida, Concentración del endeudamiento a corto plazo, etc.

Acceso

Para iniciar esta ventana, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el submenu Contabilidad y desde la sección de Informes seleccione el comando Indices Financieros.

El Informe de Indices Financieros proporciona los siguientes Filtros:

Elemento

Descripción
Abrir

Abrir un archivo donde se encuentren almacenadas formulas de Indices Financieros.
Guardar

Aquí el usuario puede almacenar todos los índices creados para consultas posteriores. Aquí se almacenan como un archivo de Indices.
Resumen Si el usuario selecciona Saldos se presentara un Informe a la fecha de corte que es la Fecha Final que el usuario ingresa, si selecciona Movimientos presentara un resultado del Indice por el periodo de tiempo que el usuario ingrese.
Fecha de Inicio

Aquí el usuario selecciona la fecha de fin de donde se va a calcular los datos para las formulas.
Fecha de Fin

Aquí el usuario selecciona la fecha de fin de donde se va a calcular los datos para las formulas.
Nuevo

Dando click en este botón se activara una línea para empezar a ingresar datos.
Quitar

Haciendo click en el botón de comandos ir a.. permite enlazarnos ha los diferentes ficheros.
Variable

Se ingresa un nombre o una letra Ejm: A– Activo Corriente ,      luego presiona el boton nuevo e ingresa la siguiente variable.
Fórmula

Una vez que ha ingresado las dos líneas anteriores de las variables y que ha escogido del campo Código las respectivas cuentas se ingresa la formula que va a realizar el calculo. Ejm:
A/B = Razón Corriente
Esta formula debe ser ingresada en una nueva línea, una vez hecho esto le da un click en el botón Actualizar o presiona la Tecla (F5) y automáticamente se le presentara la información según todos los criterios que Ud. halla seleccionado.
