Módulo: Empleados
Informe: Roles de Pago

Definición

Este informe presenta roles de pago general de empleados, con todos los rubros y valores de cada uno.

Acceso

Para acceder a esta opción, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el Submenú Empleados y desde la sección Informe seleccione el comando Roles de Pagos.

Elementos

El Informe de Roles de Pago proporciona los siguientes Filtros:

Elemento

Descripción
Calenderio 1-2 Fechas para extraer los roles.

Documento

 Botón que sirve para generar el documento de roles de pago.
Otros Reportes Botón que sirve para enlazar los reportes de Rol Resumido y Planilla del IESS.
