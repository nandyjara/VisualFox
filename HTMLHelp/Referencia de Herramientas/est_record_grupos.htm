Módulo Académico
Mantenimiento - Esquema Orgánico

Explicación

El Esquema Orgánico define la Organización actual de la Institución en cuanto a AREAS, JORNADAS, CICLOS, CARRERAS y CURSOS.  Gracias a este tipo de organización, se facilitan los procesos de matriculación, consultas estadísticas de estudiantes y más reportes y procesos del Módulo Académico.  El Esquema Orgánico permite visualizar jerárquicamente a manera de carpetas los niveles organizacionales mencionados.  Observe la siguiente figura:

Generalmente, el Esquema Orgánico se crea una sóla vez y no necesita mantenimiento a menos que la Institución decida cambiar sus políticas organizacionales, se amplíen las áreas a otros niveles educativos o cuando se emiten reformas por parte del Ministerio de Educación.

Tipos de Elementos

Cada elemento del Esquema Orgánico representa un nivel y corresponde a uno de los siguientes tipos de elementos:

AREA
JORNADA
CICLO
CARRERA
CURSO

Codificación

Cada elemento del Nivel Orgánico tiene un código que es utilizado por DOBRA® para generar varios tipos de reportes o recuperar información partiendo de un nivel dado.

El Código de cada elemento está compuesto de 5 caracteres alfanuméricos, aunque recomendamos el uso de dígitos.  El orden en se colocan los dígitos también es muy importante; a continuación se explica el uso de cada dígito en el orden que ocupan en el código:

1 AREA El Primer dígito representa el área. Ejm:

1 = JARDÍN
2 = ESCUELA
3 = COLEGIO
4 = INSTITUTO
5 = UNIVERSIDAD
2 JORNADA El Segundo dígito representa la JORNADA.  Ejm:

1 = MATUTINA
2 = VESPERTINA
3 = NOCTURNA
3 CICLO El Tercer dígito representa el Ciclo Educativo.  No todas las áreas se organizan mediante Ciclos por lo tanto; si no es aplicable el ciclo a un área determinada, se usará el 0 (cero).  Ejm:

0 = N/A
1 = Básico
2 = Diversificado
4 CARRERA El Cuarto dígito representa la carrera o especialización.  Si no es aplicable se usará el 0.  Ejm:

0 = N/A
1 = FIMA
2 = QUIBIO
3 = CONTABILIDAD
4 = INFORMÁTICA
5 CURSO Finalmente el quinto dígito debe corresponder al curso.  Ejm:

1 = Primero
2 = Segundo
3 = Tercero
4 = Cuarto, etc.

Como puede verse, antes de empezar a crear los distintos elementos del Esquema Orgánico, es importante codificar las AREAS, JORNADAS, CICLOS, CARRERAS y CURSOS que formarán el código del nivel orgánico.

Por ejemplo, El Primer Curso de Ciclo Básico de la Sección Matutina del Colegio tendría el siguiente código: 31101 basándonos en los ejemplos anteriores.

Ejecución

  1. Active el Menú de DOBRA®
  2. Seleccione el módulo Académico II
  3. En la sección de Mantenimiento seleccione la opción Esquema Orgánico.
