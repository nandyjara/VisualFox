Módulo de Punto de Ventas
Documento: Devoluciones

Definición

La devolución de punto de venta sirve para registrar los productos que el Cliente devuelve a la empresa porque no cumplen con las especificaciones solicitadas o porque están averiados.

La devolución de punto de venta representa un Crédito en la cuenta del Cliente, por esta razón, el usuario debe aplicar el monto de la devolución a una o varias de las deudas pendientes del Cliente.

Acceso

Para acceder a esta opción, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el Submenú Puntos de Venta y desde la sección Documento seleccione el comando Devoluciones.

Elementos

Elemento

Descripción

Asiento El cuadro Asiento es de "solo lectura" y muestra el número del asiento contable asociado al documento y que es generado por el sistema.
Tipo El cuadro Tipo es de "solo lectura" y muestra el Tipo de transacción.
Fecha El cuadro Fecha es "editable" y muestra fecha de la transacción.
No. de Factura El cuadro No. de factura sirve para ingresar el número de factura sobre la cual se hará la devolución.
Divisas El cuadro divisas es de "Sólo lectura" y sirve para visualizar la divisa actual.

Crear

Para crear una nuevo Documento, haga lo siguiente:

  1. Seleccione el comando Nuevo desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.

  2. El cursor se ubica en el cuadro No. Factura.  Escriba el número de la factura y luego presione la tecla TAB para mover el cursor a la sección de Detalle de Productos.

  3. Ingrese el valor de la devolución del producto.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.

  4. Seleccione la bodega de donde los productos han sido facturados.  Use la tecla ALT+FLECHA_ABAJO para seleccionar.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.  

  5. Seleccione el empaque, éste viene dado internamente según el producto de venta.  Use la tecla ALT+FLECHA_ABAJO para seleccionar.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.  

  6. Luego puede escribir una glosa con el detalle de la devolución.  Cuando este listo debe asegurarse que todos los valores estén calculados correctamente.

  7. Seleccione el comando Guardar desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.

  8. A continuación el sistema le mostrará con el nuevo número de documento asignado.
