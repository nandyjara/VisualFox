Módulo de Clientes
Informe: Saldos

Definición

Este informe presenta un listado de todos los clientes con los saldos que le adeudan a la Empresa.

Acceso

Para acceder a esta opción, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el Submenú Clientes y desde la sección Informe seleccione el comando Saldos.

Elementos

Elemento

Descripción
Carpetas Haciendo click en el icono carpetas se visualiza los grupos y zonas representado por la estructura gerárquica ya creada en el módulo de clientes.
Calendario El calendario es utilizado cuando se requiere especificar una fecha de corte para el nforme de saldos.
Vendedor El cuadro vendedor sirve para seleccionar el nombre del vendedor.
Zona El cuadro zona sirve para seleccionar la zona de donde se desea visualizar el informe.
Divisa El cuadro de divisa sirve para seleccionar la divisa.
Clase El cuadro de clase sirve para seleccionar la clase que puede ser normal, directo, subdistribuidor o mayorista.
Sucursal El cuadro de sucursal sirve para seleccionar la sucursal.
Ir a Al presionar éste icono se puede elejir el informe de fichero de registro, deudas, pagos, estado de cuenta, órdenes de compra, factura, dev. ventas, nota de crédito, nota de débito del cliente.
Otros Reportes Al presionar éste botón de comandos se puede elejir el informe de cuentas por cobrar, análisis de antiguedad de saldos, carta de pétición de pagos, facturas por cobrar del periodo.
Cuadrícula de informe de saldos

La cuadrícula del informe de saldos está compuesta por las columnas de código, nombre, saldo.  La cuadrícula del informe de saldos sirve para visualizar, aquí no se puede eliminar ni modificar.
