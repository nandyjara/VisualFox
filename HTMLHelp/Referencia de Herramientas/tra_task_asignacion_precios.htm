Módulo de Transporte
Procesos: Asignación de Precios

Definición

El proceso de asignación de precios sirve para crear el tarifario de precios para la facturación de los diferentes tipos de servicios, este tarifario puede ser general para todos o se lo puede realizar solo por cliente.

Acceso

Para acceder a esta opción, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el Submenú Transporte y desde la sección Procesos seleccione el comando Asignación de Precios.

Elementos

Elemento

Descripción
Cliente

Lista desplegable de Clientes, este elemento es opcional, si se selecciona un cliente se aplicará el precio unicamente al cliente escogido, caso contrario el precio se aplicará a todos los clientes
Carga

Lista desplegable de cargas, esta lista se alimenta de los datos ingresados en el fichero Tipos de Carga del menú mantenimiento.
Ruta

Lista desplegable de rutas, esta lista se alimenta de los datos ingresados en el fichero Rutas del menú mantenimiento.
Hono. Transporte

Lista desplegable de productos tipo servicios.
Estiba por persona

Cuadro de chequeo
Hono. Paletizada

Lista desplegable de productos tipo servicios, además se ingresa el precio de Estiba
Hono. NO Paletizada

Lista desplegable de productos tipo servicios, además se ingresa el precio de Estiba
Cuadrícula

Se ingresan datos de: Peso (Kg) Desde, Peso (Kg) hasta, Precio (Kg) , Vol (m3) desde, Vol (m3) hasta, Precio (m3).

Crear

Para crear un nuevo proceso de asignación de precios haga lo siguiente:

  1. El cursor se ubica en el cuadro cliente. Seleccione el cliente con las teclas CTRL+Flecha abajo.
  2. A continuación presionar la tecla TAB para seleccionar la carga de la lista desplegable. Presione  la tecla TAB para movilizarse al cuadro de Ruta, seleccionar la ruta y el Honorario por transporte
  3. Adicional debe ingresar los datos de honorarios paletizada y no paletizada.

  4. Para registrar los datos de detalles de la cuadrícula, se debe agregar nuevas filas en la cuadrícula de detalles.  Para agregar una nueva fila haga un click en el botón () de la sección detalles de documento o presione la tecla ALT+N.  Aparece una nueva fila en la cuadrícula y el cursor se ubica en la primera celda de la fila.

  5. Ingrese los datos de Código de pesos (kg) desde-hasta, precio(kg), volúmen(m3) desde-hasta y precio(m3) .

  6. Si desea ingresar una nuevo detalle regrese al paso 4.  Si desea quitar una fila de la cuadrícula de detalle, haga un click en la fila correspndiente y luego haga un click el botó n () de la sección Detalle o presione la tlecla ALT+Q.  

  7. Seleccione el comando Guardar desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.
