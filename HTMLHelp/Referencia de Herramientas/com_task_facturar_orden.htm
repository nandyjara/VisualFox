Módulo de Compras
Proceso: Facturar Orden

Definición

El Proceso de Facturar Orden sirve para que el usuario después que ha ingresado al Inventario los productos de la Orden de Compra realice la Factura para generar una deuda con el Proveedor.

Acceso

Para acceder a esta opción, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el Submenú Compras y desde la sección Proceso seleccione el comando Facturar Orden.

Elementos

Elemento

Descripción
Orden de compra El cuadro orden de compra sirve para ingresar el numero de la orden de compra.
Tipo El cuadro Tipo es de "solo lectura" y muestra el Tipo de transacción.
Fecha El cuadro Fecha es de "solo lectura" y muestra fecha de la transacción.
Divisas El cuadro divisa es de "solo lectura" sirve para seleccionar la divisa ó se visualiza la divisa según el tipo de cambio.
Cambio El cuadro Cambio es de "sólo lectura" y sirve para visualizar el tipo de cambio.
Fact. Proveedor El cuadro de factura del proveedor sirve para ingresar el numero de factura del proveedor.
Proveedor El cuadro Proveedor es de "solo lectura" sirve para seleccionar el proveedor a quién se le factura la orden.
Término El cuadro Término es de "solo lectura" y sirve para seleccionar la forma de pago pueden ser  a 5, 15, 20 días.
División El cuadro de División es de "sólo lectura" sirve para seleccionar la división o línea de negocio a la que pertenece el documento.
Detalle de Facturar Orden La sección de Detalle de Facturar Orden está compuesta por los cuadros código, nombre, cantidad, costo, empaque, desc%, desc, total; Estos cuadros sirven para visualizar y editar ciertos valores de los productos de de la factura.
Nota que se Imprime en la Orden Este cuadro es "editable" y sirve para ingresar una descripción impresa de la transacción.

Crear

Para crear un nuevo proceso, haga lo siguiente:

  1. El cursor se ubica en el cuadro Orden de Compra, ingrese el número correspondiente a la orden.
  2. Luego ingrese el número de la factura del proveedor.

  3. A continuación se visualiza todos los productos que pertenecen al proceso de facturar orden.  Se puede editar solo la columna de Cantidad, que no puede ser mayor al ingreso, caso contrario el sistema genera un error de ingreso de datos. En la columna de Ingreso se encuentra la cantidad de mercadería ingresada o recibida ya en bodega; este valor es automáticamente generado por DOBRA.

  4. Seleccione el comando Guardar desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.

  5. A continuación el sistema le mostrará con el nuevo número de documento asignado.
