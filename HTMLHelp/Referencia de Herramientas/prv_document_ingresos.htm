Módulo de Proveeduria
Documento: Nota de Ingreso

Definición

La Nota de Ingreso de Productos sirve para registrar una entrada de mercaderías en alguna Bodega de suministros para la empresa por un concepto distinto al de una compra

Acceso

Para acceder a esta opción, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el Submenú Proveduria y desde la sección Documento seleccione el comando Nota de Ingreso.

Elementos

Elemento

Descripción
Buscar Sirve para buscar el documento por número.
Estado El cuadro estado es de "solo lectura" y muestra información acerca del estado actual del documento Ej: ANULADO, APROBADO, ETC.
ID El cuadro ID es de "solo lectura" y muestra el número de identificación relacional del documento que es generado por el sistema.
Número El cuadro Numero es de "solo lectura" y muestra número secuencial del documento que es generado por el sistema.
Asiento El cuadro Asiento es de "solo lectura" y muestra el número del asiento contable asociado al documento y que es generado por el sistema.
Tipo El cuadro Tipo es de "solo lectura" y muestra el Tipo de transacción.
Fecha El cuadro Fecha es "editable" y muestra fecha de la transacción.
División El cuadro de lista de división sirve para seleccionar la división o línea de negocio a la que pertenece el documento
O. Compra El cuadro de Orden de Compra sirve para visualizar el número de la misma "solo lectura".
Concepto El cuadro Concepto es "editable" y sirve para ingresar una descripción corta o abreviada del motivo de la transacción.
Detalle de los Productos La sección de Detalle Ingresos de los Productos que Ingresan está compuesto por una cuadrícula de productos que tiene las columnas de código, nombre, bodega, cantidad, divisa, costo, costo base, saldo basetotal y además de los botones de comandos Nuevo y Quitar que sirve para agregar o quitar filas en la cuadrícula.
La sección de los Productos cuenta además con el total.
Justificación Contable La sección de Detalle Justificación Contable está compuesta por una cuadrícula de rubros que tienen las siguientes columnas: código, rubro, detalle, divisa, haber, valor base.
La sección de Detalle Justificación Contable sirve para registrar el asiento de contrapartida, y además cuenta con el total debe, total haber, diferencia.
Observaciones El cuadro observaciones es "editable" y sirve para ingresar una descripción completa de la transacción.

Crear

Para crear una nuevo Documento, haga lo siguiente:

  1. Seleccione el comando Nuevo desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.

  2. El cursor se ubica en el cuadro Proveedor.  Escriba el nombre del proveedor y luego presione la tecla TAB para mover el el cursor al cuadro Fact.Prov.

  3. Ingrese la factura del proveedor.  Acontinuación presione la tecla TAB para mover el cursor al cuadro término.

  4. Seleccione el témino de pago al que corresponde la factura.  Puede usar las teclas ALT+FLECHA_ABAJO o CTRL+FLECHA_ABAJO.  Acontinuación presione la tecla TAB para mover el cursor al cuadro seguro.

  5. Ingrese el seguro, puede ser propio o del proveedor..  Acontinuación presione la tecla TAB para mover el cursor al cuadro pago.

    Ingrese el pago, puede ser giro directo, cheque, tarjeta.  Acontinuación presione la tecla TAB para mover el cursor al cuadro empaque.

  6. Ingrese el empaque.  Acontinuación presione la tecla TAB para mover el cursor al cuadro marca y números.

  7. Ingrese la marca y números.  Acontinuación presione la tecla TAB para mover el cursor al cuadro país de origen.

  8. Ingrese el país de origen de donde sale el embarque.  Acontinuación presione la tecla TAB para mover el cursor al cuadro negociación.

  9. Ingrese la negociación, puede ser FOB (flete a su puerto de destino), CIF (flete a su puerto de destino con seguro incluido).  Acontinuación presione la tecla TAB para mover el cursor al cuadro transporte.

  10. Ingrese el transporte, puede ser aéreo, terrestre, marítimo.  Acontinuación presione la tecla TAB para mover el cursor al cuadro peso neto aproximado.

  11. Ingrese el peso neto aproximado de la mercadería.  Acontinuación presione la tecla TAB para mover el cursor a la sección de detalle de pedido.

  12. Para registrar los productos que serán descritos en el pedido de importación, se debe agregar nuevas filas en la cuadrícula de Detalle de Pedido.  Para agregar una nueva fila haga un click en el botón () de la sección Detalle de Pedido o presione la tecla ALT+N.  Aparece una nueva fila en la cuadrícula y el cursor se ubica en la primera celda de la fila.

  13. Seleccione el código del producto.  Use la tecla ALT+FLECHA_ABAJO y seleccione el producto.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.  

  14. Ingrese la cantidad del producto.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.

  15. Seleccione el empaque, éste viene dado internamente según el producto de venta.  Use la tecla ALT+FLECHA_ABAJO para seleccionar.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.  

  16. Ingrese el precio del producto.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.

  17. Si desea ingresar una nuevo Detalle de Pedido regrese al paso 12.  Si desea quitar una fila de la cuadrícula de pedido, haga un click en la fila correspondiente y luego haga un click el botón () de la sección Detalle de Pedido ó presione la tecla ALT+Q.  

  18. Luego puede escribir una glosa con el detalle del pedido de importación.  Cuando este listo debe asegurarse que todos los valores estén calculados correctamente.

  19. Seleccione el comando Guardar desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.

  20. Acontinuación el sistema le mostrará con el nuevo número de documento asignado.
