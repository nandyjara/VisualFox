Módulo: Ventas
Documento:  Orden de Despachos

Definición

El Documento de Orden de Despacho sirve para crear la orden que será entregada al bodeguero para ser despachada.

Acceso

Para acceder a esta opción, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el Submenú Ventas y desde la sección Documentos seleccione el comando Orden de Despachos.

Elementos

Elemento

Descripción
Buscar Sirve para buscar el documento por número.
Estado El cuadro estado es de "solo lectura" y muestra información acerca del estado actual del documento Ej: ANULADO, APROBADO, ETC.
ID El cuadro ID es de "solo lectura" y muestra el número de identificación relacional del documento que es generado por el sistema.
Número El cuadro Numero es de "solo lectura" y muestra número secuencial del documento que es generado por el sistema.
Asiento El cuadro Asiento es de "solo lectura" y muestra el número del asiento contable asociado al documento y que es generado por el sistema.
Tipo El cuadro Tipo es de "solo lectura" y muestra el Tipo de transacción.
Fecha El cuadro Fecha es de "editable" y muestra fecha de la transacción.
División El cuadro de división sirve para seleccionar la división o línea de negocio a la que pertenece el documento.
Entrega

El cuadro entrega sirve para seleccionar la empresa a quien se hace el despacho.
Cliente  El cuadro Cliente es "editable" sirve para seleccionar el nombre del cliente a quien los productos van a ser despachados.
Persona que Envía  El cuadro Persona que Envía es "editable" sirve para seleccionar el nombre de quién envía los productos.
Persona que Recibe  El cuadro Persona que Recibe es "editable" sirve para ingresar el nombre del cliente quien recibe los productos.
concepto

El cuadro concepto sirve para ingresar el motivo del despacho. 
Referencia

El cuadro Referencia sirve para ingresar información adicional del producto para el despacho. 
Detalle de Productos 

La sección de Detalle de Productos está compuesto por una cuadrícula de productos que tienen las siguientes columnas: código, nombre, order, empaque, cantidad, empaque, total, back-O.
La sección de detalle de Productos sirve para registrar los diferentes productos que van a ser entregados al cliente, cuenta además con el botón de Quita con el que se puede restar los productos que no irán en el documento de despacho. 

Observaciones

El cuadro observaciones es "editable" y sirve para ingresar una descripción completa de la transacción.

Crear

Para crear una nuevo documento, haga lo siguiente:

  1. Seleccione el comando Nuevo desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.

  2. El cursor se ubica en el cuadro división.  Elija la división correspondiente y luego presione la tecla TAB para mover el el cursor al cuadro entrega.

  3. Seleccione la entrega, el número corespondiente a la nota de entrega .  Puede usar las teclas ALT+FLECHA_ABAJO o CTRL+FLECHA_ABAJO.  Acontinuación presione la tecla TAB para mover el cursor al cuadro cliente.
  4. Seleccione el cliente a quién se despacha los productos.  Puede usar las teclas ALT+FLECHA_ABAJO o CTRL+FLECHA_ABAJO.  Acontinuación presione la tecla TAB para mover el cursor al cuadro persona envía.
  5. Seleccione la persona que envía los productos.  Puede usar las teclas ALT+FLECHA_ABAJO o CTRL+FLECHA_ABAJO.  Acontinuación presione la tecla TAB para mover el cursor al cuadro persona recibe.
  6. Seleccione la persona que recibe los productos.  Puede usar las teclas ALT+FLECHA_ABAJO o CTRL+FLECHA_ABAJO.  Acontinuación presione la tecla TAB para mover el cursor al cuadro concepto.
  7. Escriba el concepto del despacho y luego presione la tecla TAB para mover el el cursor al cuadro referencia.
  8. Escriba la referencia , información adicional del despacho y luego presione la tecla TAB para mover el cursor a la seción de Detale de Productos. 

  9. Para registrar los productos que serán descritos en la orden de despacho, se debe agregar nuevas filas en la cuadrícula de Detalle de Productos.  Para agregar una nueva fila haga un click en el botón () de la sección Detalle de Productos o presione la tecla ALT+N.  Aparece una nueva fila en la cuadrícula y el cursor se ubica en la primera celda de la fila.

  10. Seleccione el código del producto.  Use la tecla ALT+FLECHA_ABAJO y seleccione el producto.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.  

  11. Escriba la order, la cantidad de pedido que hace el cliente y luego presione la tecla TAB para mover el cursor a la siguiente celda.

  12. Ingrese la cantidad de entrega inmediata del producto.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.

  13. Seleccione el empaque, éste viene dado internamente según el producto de venta.  Use la tecla ALT+FLECHA_ABAJO para seleccionar.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.  

  14. Ingrese el Back_O, la cantidad del producto pendiente por entregar.  A continuación presione la tecla TAB para mover el cursor a la siguiente celda.

  15. Si desea ingresar una nuevo Detalle de Productos regrese al paso 9.  Si desea quitar una fila de la cuadrícula de productos, haga un click en la fila correspondiente y luego haga un click el botón () de la sección Detalle de Productos ó presione la tecla ALT+Q.  

  16. Luego puede escribir una glosa con el detalle de la orden de despachos.  Cuando este listo debe asegurarse que todos los valores estén calculados correctamente.

  17. Seleccione el comando Guardar desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.

  18. Acontinuación el sistema le mostrará con el nuevo número de documento asignado.
