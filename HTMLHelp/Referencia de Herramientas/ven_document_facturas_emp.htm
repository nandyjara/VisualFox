Módulo: Ventas
Documento: Factura de Ventas de Empleados

Definición

La Factura de Venta a Empleados registra una deuda del Empleado con la Empresa por compra de un producto. 

Integración

Acceso

Para acceder a esta opción, haga lo siguiente:

  1. Abra el menú Herramientas de DOBRA®.
  2. Despliegue el Submenú Ventas y desde la sección Documento seleccione el comando Factura a Crédito de Empleados.

Elementos

Elemento

Descripción
Buscar Sirve para buscar el documento por número.
Estado El cuadro estado es de "solo lectura" y muestra información acerca del estado actual del documento Ej: ANULADO, APROBADO, ETC.
ID El cuadro ID es de "solo lectura" y muestra el número de identificación relacional del documento que es generado por el sistema.
Número El cuadro Numero es de "solo lectura" y muestra número secuencial del documento que es generado por el sistema.
Asiento El cuadro Asiento es de "solo lectura" y muestra el número del asiento contable asociado al documento y que es generado por el sistema.
Tipo El cuadro Tipo es de "solo lectura" y muestra el Tipo de transacción.
Fecha El cuadro Fecha es de "editable" y muestra fecha de la transacción.
Divisa El cuadro de divisa sirve para seleccionar la divisa ó visualizar ésta divisa según el tipo de cambio.
Cambio

El cuadro Cambio es de "Sólo lectura" y sirve para visualizar el tipo de cambio.
División

El cuadro de División sirve para seleccionar la división o línea de negocio a la que pertenece el documento.
Empleado

El cuadro de Empleado sirve para seleccionar el empleado a quién se le hace la factura.
Cédula

El cuadro de Cédula sirve para ingresar el número de cédula del empleado.
No. Guía

El cuadro de No. Guía sirve para ingresar el No. guía remisión del transporte.
Transporte

El cuadro de Transporte sirve para seleccionar medio de transporte.
No. factura

El cuadro de No. factura sirve para ingresar el número de la factura de empellados.
Términos

El cuadro de Términos sirve para seleccionar el plazo 15, 20, 45 días en que se realiza la orden de venta.
Vendedor

El cuadro de Vendedor sirve para seleccionar el vendedor quién realiza la factura de empleados.
Detalle de Productos

La sección de Detalle de Productos está compuesta por los cuadros código, nombre, cantidad, empaque, total, lista, precio, desc%, desc, extendido; Estos cuadros son editables y sirven para ingresar los datos de los productos. Además cuenta con el botón de comando nuevo y quitar el cual sirve para quitar o agregar una nueva fila en la cuadrícula.
Forma de Pago

La sección de Forma de Pago está compuesta por los cuadros fecha, Vencimiento, detalle, cuota; Estos cuadros son editables también sirven para visualizar los valores repartidos de la forma de pago. Además cuenta con el botón de comando Generar Pagos el cual sirve para visualizar los pagos generados por el sistema con las fechas inicial y de vencimiento de las respectivas cuotas.
Nota

Este cuadro Nota es "editable" y sirve para ingresar un comentario  de la transacción. 

Crear

Para crear una nuevo Documento, haga lo siguiente:

  1. Seleccione el comando Nuevo desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.

  2. Seleccione el empleado, Puede usar las teclas ALT+FLECHA_ABAJO o CTRL+FLECHA_ABAJO.  A continuación presione la tecla TAB para mover el cursor al cuadro cédula.

  3. Escriba el número de cédula y luego presione la tecla TAB para mover el cursor al cuadro No. Guía.

  4. Escriba el número de Guía de Remisión y luego presione la tecla TAB para mover el cursor al cuadro transporte.

  5. Seleccione el transporte, Puede usar las teclas ALT+FLECHA_ABAJO o CTRL+FLECHA_ABAJO.  A continuación presione la tecla TAB para mover el cursor al cuadro No. Factura.

  6. Escriba el número de factura y luego presione la tecla TAB para mover el cursor al cuadro término.

  7. Seleccione el término puede ser 15,20,45 días, Puede usar las teclas ALT+FLECHA_ABAJO o CTRL+FLECHA_ABAJO.  A continuación presione la tecla TAB para mover el cursor al cuadro vendedor.

  8. Seleccione el vendedor, Puede usar las teclas ALT+FLECHA_ABAJO o CTRL+FLECHA_ABAJO.  A continuación presione la tecla TAB para mover el cursor al cuadro división.

  9. Seleccione la división, Puede usar las teclas ALT+FLECHA_ABAJO o CTRL+FLECHA_ABAJO.  A continuación presione la tecla TAB para mover el cursor a la sección de Detalle de Productos.

  10. Para registrar los Productos debe agregar nuevas filas en la cuadrícula de productos.  Para agregar una nueva fila haga un click en el botón () de la sección productos o presione la tlecla ALT+N.  Aparece una nueva fila en la cuadrícula y el cursor se ubica en la primera celda de la fila.

  11. Seleccione el código del producto .  Puede usar las teclas ALT+FLECHA_ABAJO o CTRL+FLECHA_ABAJO.  A continuación presione la tecla tab para mover el cursor a la siguiente celda.

  12. Haga click en () y aparecerá una ventana de diálogo en la cual podrá escribir una explicación más detallada acerca del producto y presione la tecla TAB para mover el cursor a la siguiente celda.

  13. Seleccione la bodega.  Puede usar las teclas ALT+FLECHA_ABAJO o CTRL+FLECHA_ABAJO.  A continuación presione la tecla tab para mover el cursor a la siguiente celda.

  14. Escriba la cantidad y luego presione la tecla TAB para mover el el cursor a la siguiente celda.

  15. Seleccione empaque, que viene dado en unidades.  Puede usar las teclas ALT+FLECHA_ABAJO o CTRL+FLECHA_ABAJO.  A continuación presione la tecla tab para mover el cursor a la siguiente celda.

  16. Seleccione el precio, que puede ser de contado, 2 créditos, etc.  Puede usar las teclas ALT+FLECHA_ABAJO o CTRL+FLECHA_ABAJO.  A continuación presione la tecla tab para mover el cursor a la siguiente celda.

  17. Ingrese el %descuento y luego presione la tecla TAB para mover el el cursor a la siguiente celda.

  18. Si desea ingresar un nuevo producto regrese al paso 10.  Si desea quitar una fila de la cuadrícula de productos que ingresan, haga un click en la fila correspondiente y luego haga un click el botón () de la sección Detalle de Productos que Ingresan o presione la tlecla ALT+Q.  Luego presione la tecla TAB para mover el cursor a la sección de forma de pago

  19. Haga click en el spiner (incrementador) ( ) para elegir en cuantas cuotas se desea cancelar la deuda .  

  20. Haga click en el botón de comandos ( ).  A continuación Ud. Puede visualizar los pagos generados repartidos en cuotas iguales por el sistema.  A continuación presione la tecla tab para mover el cursor a la sección de pagos generados para repartir valores manualmente si se desea.

  21. Luego puede escribir una glosa con el detalle de factura de empleados.  Cuando este listo debe asegurarse que no exista ningún error los valores calculados por el sistema.

  22. Seleccione el comando Guardar desde el Menú Archivo o haga un click en el botón () de la Barra de Herramientas.

  23. A continuación el sistema le mostrará con el nuevo número de documento asignado.
