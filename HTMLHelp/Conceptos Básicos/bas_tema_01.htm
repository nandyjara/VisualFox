Qué es DOBRA ?

DOBRA es un SISTEMA INTEGRADO DE INFORMACIÓN EMPRESARIAL CLIENTE/SERVIDOR.

DOBRA es un sistema comercial que proporciona una solución integral a los requierimientos financieros, administrativos y operativos de la empresa.  DOBRA es un Sistema multiusuario que permite el trabajo cooperativo en la red, proporcionando herramientas en pantalla para el registro y contabilización de transacciones, para el ingreso y mantenimiento de datos o parámetros empresariales, para la ejecución de procesos especiales y para la generación de una gran variedad de informes a todo nivel.

DOBRA es un producto de software diseñado para funcionar en los sistemas operativos Windows de Microsoft (desde la versión Windows95 en adelante).   DOBRA se instala en la computadora de cada usuario y desde allí se conecta a una Base de Datos para actualizar y recuperar información.
